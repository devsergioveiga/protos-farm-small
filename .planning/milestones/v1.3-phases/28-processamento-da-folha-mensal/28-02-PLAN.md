Create `salary-advances.service.ts`:

**`createAdvance(rls: RlsContext, input: CreateAdvanceInput): Promise<SalaryAdvance>`**

1. Fetch employee with current salary (via EmployeeSalaryHistory latest or EmployeeContract.salary).
2. Validate: employee.status must be 'ACTIVE'. Throw SalaryAdvanceError('EMPLOYEE_NOT_ACTIVE') if not.
3. Validate limit: get org-level max percent (default 40%). If `amount > salary * maxPercent / 100`, throw SalaryAdvanceError('LIMIT_EXCEEDED', `Adiantamento excede o limite de ${maxPercent}% do salário`).
4. Check duplicate: if SalaryAdvance exists for same employeeId + referenceMonth, throw SalaryAdvanceError('DUPLICATE_ADVANCE').
5. Use `prisma.$transaction`:
   - Create SalaryAdvance record with referenceMonth as `new Date(input.referenceMonth + '-01')`, advanceDate as `new Date(input.advanceDate)`.
   - Create Payable with `tx.payable.create({ data: { organizationId: rls.organizationId, farmId: employee.primaryFarmId || employee.farms[0]?.farmId, supplierName: employee.name, category: 'PAYROLL', description: 'Adiantamento salarial ${monthLabel} - ${employee.name}', totalAmount: amount, dueDate: new Date(input.advanceDate), status: 'PENDING', originType: 'SALARY_ADVANCE', originId: advance.id }})`.
   - Update advance with payableId.
6. Return created advance.

**`createBatchAdvances(rls: RlsContext, input: BatchAdvanceInput): Promise<{ batchId: string; count: number; advances: SalaryAdvance[] }>`**

1. Fetch all active employees for the org.
2. For each, check if SalaryAdvance already exists for referenceMonth — skip if yes.
3. Calculate amount = salary \* (input.percentOfSalary || 40) / 100, rounded to 2dp.
4. Generate batchId = `batch-${Date.now()}`.
5. For each eligible employee, call createAdvance logic (inside single transaction per employee, not one big tx).
6. Return { batchId, count, advances }.

**`listAdvances(rls: RlsContext, filters: { referenceMonth?: string; employeeId?: string; page?: number; limit?: number }): Promise<{ data: SalaryAdvance[]; total: number }>`**

1. Build Prisma where clause with organizationId from rls, optional referenceMonth filter, optional employeeId.
2. Include employee name and payable status in select.
3. Paginate with skip/take.

**`generateAdvanceReceiptPdf(rls: RlsContext, advanceId: string): Promise<Buffer>`**

1. Fetch advance with employee and org data.
2. Use pdfkit (dynamic import) with A4 size, margin 50.
3. Layout: header (org name, CNPJ), title "RECIBO DE ADIANTAMENTO SALARIAL", employee data (name, CPF, cargo), amount (R$ formatted), date, competencia, signature line.
4. Return Buffer.

Create `salary-advances.routes.ts`:

```typescript
import { Router } from 'express';
import { requireAuth, requirePermission } from '../../shared/rbac/middleware';
import * as service from './salary-advances.service';

const router = Router();
router.use(requireAuth);

// POST /org/:orgId/salary-advances — create individual advance
router.post('/', requirePermission('payroll-params', 'write'), async (req, res, next) => {
  try {
    const orgId = req.params.orgId as string;
    const advance = await service.createAdvance(
      { organizationId: orgId, userId: (req as any).user.id },
      req.body,
    );
    res.status(201).json(advance);
  } catch (err) {
    next(err);
  }
});

// POST /org/:orgId/salary-advances/batch — create batch advances per D-07
router.post('/batch', requirePermission('payroll-params', 'write'), async (req, res, next) => {
  try {
    const orgId = req.params.orgId as string;
    const result = await service.createBatchAdvances(
      { organizationId: orgId, userId: (req as any).user.id },
      req.body,
    );
    res.status(201).json(result);
  } catch (err) {
    next(err);
  }
});

// GET /org/:orgId/salary-advances — list with filters
router.get('/', requirePermission('payroll-params', 'read'), async (req, res, next) => {
  try {
    const orgId = req.params.orgId as string;
    const result = await service.listAdvances(
      { organizationId: orgId, userId: (req as any).user.id },
      {
        referenceMonth: req.query.referenceMonth as string | undefined,
        employeeId: req.query.employeeId as string | undefined,
        page: req.query.page ? Number(req.query.page) : undefined,
        limit: req.query.limit ? Number(req.query.limit) : undefined,
      },
    );
    res.json(result);
  } catch (err) {
    next(err);
  }
});

// GET /org/:orgId/salary-advances/:id/receipt — download receipt PDF per D-07
router.get('/:id/receipt', requirePermission('payroll-params', 'read'), async (req, res, next) => {
  try {
    const orgId = req.params.orgId as string;
    const id = req.params.id as string;
    const pdf = await service.generateAdvanceReceiptPdf(
      { organizationId: orgId, userId: (req as any).user.id },
      id,
    );
    res.setHeader('Content-Type', 'application/pdf');
    res.setHeader('Content-Disposition', `attachment; filename="recibo-adiantamento-${id}.pdf"`);
    res.send(pdf);
  } catch (err) {
    next(err);
  }
});

export default router;
```
