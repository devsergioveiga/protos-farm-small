Create `payroll-calculation.service.ts` with two exported functions:

**`calculateEmployeePayroll(input: EmployeePayrollInput, referenceMonth: Date, engineParams: EngineParams): EmployeePayrollResult`**

Where `EngineParams` includes `inssBrackets`, `irrfTable`, `fgtsCeiling`, `salaryFamilyLimits`, `ratPercent` (from payroll-tables).

Implementation sequence:

1. Calculate pro-rata if admission is within referenceMonth: `proRataDays = daysInMonth - admissionDay + 1; proRataFactor = proRataDays / daysInMonth; adjustedSalary = baseSalary * proRataFactor`
2. Calculate hourly rate: `hourlyRate = adjustedSalary / 220` (standard CLT monthly hours)
3. Calculate overtime50: `(timesheetData.totalOvertime50 / 60) * hourlyRate * 1.5`
4. Calculate overtime100: `(timesheetData.totalOvertime100 / 60) * hourlyRate * 2.0`
5. Calculate DSR on overtime: Use `date-holidays` (constructor: `new Holidays('BR', stateCode)`) to get holidays in referenceMonth. Count Sundays + holidays = restDays. workDays = daysInMonth - restDays. `dsrValue = (overtime50 + overtime100) * (restDays / workDays)`. Import Holidays from `date-holidays`.
6. Call `calculateRuralNightPremium({ nightMinutes: timesheetData.totalNightMinutes, hourlyRate })` for nightPremium.
7. Call `calculateSalaryFamily({ grossSalary: adjustedSalary, dependentsUnder14 })` for salaryFamily.
8. Sum provisions: `grossSalary = adjustedSalary + overtime50 + overtime100 + dsrValue + nightPremium + salaryFamily.amount + otherProvisions`
9. Call `calculateINSS(grossSalary, inssBrackets, ceiling)` — get inssAmount.
10. Call `calculateIRRF({ grossSalary, inssContribution: inssAmount, dependents: dependentsCount, alimony: alimonyAmount })` — get irrfAmount.
11. Call `calculateRuralUtilityDeductions({ salary: baseSalary, housingProvided, foodProvided })` — get housing/food deductions.
12. VT deduction: `vtDeduction = adjustedSalary * vtPercent / 100` (capped at 6% per CLT).
13. `netSalary = grossSalary - inssAmount - irrfAmount - vtDeduction - housingDeduction - foodDeduction - pendingAdvances - otherDeductions`
14. Call `calculateFGTS(grossSalary)` — employer info.
15. `inssPatronal = grossSalary * 0.20 + grossSalary * ratPercent / 100` (INSS patronal 20% + RAT).
16. Build `lineItems` array with each component as a line item (code, description, reference, type, value).

All arithmetic uses `Decimal` from `decimal.js`. Round to 2dp with `Decimal.ROUND_HALF_UP` at each step.

**`calculateThirteenthSalary(input: ThirteenthSalaryInput, engineParams: EngineParams): EmployeePayrollResult`**

For FIRST parcel:

- `proportional = baseSalary * monthsWorked / 12`
- `gross = proportional / 2` (half of annual)
- No INSS, no IRRF deductions
- `net = gross`

For SECOND parcel:

- `grossBase = (baseSalary + avgOvertime50 + avgOvertime100 + avgNightPremium) * monthsWorked / 12`
- Calculate INSS on grossBase
- Calculate IRRF on (grossBase - INSS - dependents)
- `net = grossBase - INSS - IRRF - firstParcelAmount`

Build lineItems for both parcel types.
