**hooks/useSafetyCompliance.ts:**
- `fetchSummary(farmId?: string)` — GET `/safety-compliance/summary?farmId=...` → ComplianceSummary
- `fetchNonCompliantEmployees(query: ComplianceDashboardQuery)` — GET `/safety-compliance/employees?farmId=...&pendingType=...&search=...&page=...&limit=...` → { data: EmployeeCompliance[], total: number }
- `fetchEmployeeCompliance(employeeId: string)` — GET `/safety-compliance/employees/${employeeId}` → EmployeeCompliance
- `exportCsv(farmId?: string)` — GET `/safety-compliance/report/csv?farmId=...` as blob, trigger download with `<a href=... download="conformidade-nr31.csv">`
- `exportPdf(farmId?: string)` — GET `/safety-compliance/report/pdf?farmId=...` as blob, open in new tab
- State: summary, employees, loading, error
- Accepts farmId parameter from FarmContext, refetches when farmId changes

**SafetyKpiCard.tsx:**

```tsx
import type { LucideIcon } from 'lucide-react';

interface Props {
  label: string;
  value: number;
  icon: LucideIcon;
  suffix?: string; // e.g. "%" for percentage
  borderColor?: string; // left border accent color
}

export function SafetyKpiCard({ label, value, icon: Icon, suffix, borderColor }: Props) {
  return (
    <div
      className="safety-kpi-card"
      style={
        borderColor
          ? { borderLeftColor: borderColor, borderLeftWidth: '4px', borderLeftStyle: 'solid' }
          : undefined
      }
    >
      <div className="safety-kpi-card__icon">
        <Icon size={24} aria-hidden="true" />
      </div>
      <div className="safety-kpi-card__content">
        <span className="safety-kpi-card__value">
          {value.toLocaleString('pt-BR')}
          {suffix || ''}
        </span>
        <span className="safety-kpi-card__label">{label}</span>
      </div>
    </div>
  );
}
```

- Value: DM Sans 700, 30px (var(--text-2xl)), color neutral-800
- Label: Source Sans 3 600, 14px (var(--text-sm)), color neutral-500
- Card: neutral-100 bg, border-radius 8px, padding 20px, hover translateY(-2px) + shadow-lg with 100ms ease-out transition
- Responsive: grid 4 cols on lg, 2x2 on md, 1 col on sm

CSS for the card goes into SafetyDashboardPage.css (or a shared file).
