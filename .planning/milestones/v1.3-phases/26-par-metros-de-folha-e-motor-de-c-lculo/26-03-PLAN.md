**Create `payroll-rubricas.service.ts`:**

   - Filter by `search` (name or code ILIKE)
   - Order by: system rubricas first, then by name
   - Use `withRlsContext(prisma, orgId, async (tx) => ...)`

2. `getById(orgId: string, id: string)` — single rubrica, validate orgId ownership

3. `create(orgId: string, data: CreateRubricaInput, userId: string)` — create rubrica:

   - Use `withRlsContext`

Service object `payrollTablesService` with methods:

   - Order by effectiveFrom DESC (most recent first)
   - Include `brackets` (ordered by `order` ASC) and `scalarValues`
   - Use `withRlsContext` for org-specific, but also query global tables with direct prisma (no RLS for null orgId)

2. `getEffective(orgId: string, tableType: LegalTableType, competenceDate: Date)` — get the effective table for a specific competence:
   - First look for org-specific table: `organizationId = orgId, tableType, effectiveFrom <= competenceDate`, order by effectiveFrom DESC, take first
   - If not found, fall back to global table: `organizationId = null, tableType, effectiveFrom <= competenceDate`, order by effectiveFrom DESC, take first
   - Include brackets and scalarValues
   - Return null if no table found (caller should handle)

3. `create(orgId: string, data: CreateLegalTableInput, userId: string)` — create new table version:
   - Validate: effectiveFrom must be first day of month (day === 1)
   - Validate: no duplicate table for same orgId + tableType + effectiveFrom
   - Create table with nested brackets and scalarValues
   - Use `withRlsContext`

4. `getById(orgId: string, id: string)` — get single table by ID with brackets and scalars
