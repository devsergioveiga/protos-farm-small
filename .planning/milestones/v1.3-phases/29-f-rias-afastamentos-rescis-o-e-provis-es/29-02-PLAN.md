    **termination-calculation.service.ts — pure functions, no DB access:**

    `calcNoticePeriodDays(admissionDate: Date, terminationDate: Date): number`
    - yearsCompleted = floor((terminationDate - admissionDate) / 365.25 days)
    - return min(30 + max(0, yearsCompleted - 1) * 3, 90)

    `FGTS_PENALTY` map: WITHOUT_CAUSE→0.40, MUTUAL_AGREEMENT→0.20, others→0.00

    `calculateTermination(input: TerminationInput, params: EngineParams): TerminationResult`
    Following RESEARCH.md Pattern 2 exactly:
    1. balanceSalary = lastSalary * daysWorkedLastMonth / daysInLastMonth (use getUTCDate per Phase 28 decision)
    2. noticePeriodDays: WITHOUT_CAUSE → calcNoticePeriodDays; MUTUAL_AGREEMENT → calcNoticePeriodDays / 2 (round up); others → 0
    3. noticePay: COMPENSATED + (WITHOUT_CAUSE|MUTUAL_AGREEMENT) → lastSalary * noticePeriodDays / 30; else 0
    4. thirteenthProp = (lastSalary + avgOvertime + avgNight) * monthsThirteenth / 12
    5. vacationVested = dailyRate * vacationVestedDays (no 1/3 in this line)
    6. vacationProp = dailyRate * vacationPropDays (no 1/3 in this line)
    7. vacationBonus = (vacationVested + vacationProp) / 3  (the 1/3 constitutional bonus)
    8. fgtsPenalty = fgtsBalance * FGTS_PENALTY[terminationType]
    9. totalGross = balanceSalary + noticePay + thirteenthProp + vacationVested + vacationProp + vacationBonus
    10. INSS = calculateINSS(totalGross, params.inssBrackets, params.inssCeiling)
    11. IRRF = calculateIRRF({grossSalary: totalGross, inssContribution: inss.contribution, dependentCount, ...params})
    12. totalNet = totalGross - INSS - IRRF + fgtsPenalty (FGTS penalty is not taxable income)
    13. paymentDeadline = terminationDate + 10 calendar days

    All arithmetic uses Decimal from decimal.js with `.toDecimalPlaces(2, Decimal.ROUND_HALF_UP)`.

    **employee-terminations.service.ts:**
    - `processTermination(input: CreateTerminationInput, ctx)`:
      1. Load employee with admissionDate, current salary (from EmployeeSalaryHistory latest), dependent count
      2. Query VacationAcquisitivePeriod to compute vacationVestedDays and vacationPropDays
      3. Query PayrollRunItem last 12 months to compute avgOvertime and avgNight
      4. Compute monthsThirteenth (months worked in current year where >=15 days counts as full month)
      5. Estimate fgtsBalance = sum of 8% * gross salary for all months employed (from PayrollRunItem)
      6. Load EngineParams (same pattern as payroll-runs)
      7. Call calculateTermination(input, params)
      8. Create EmployeeTermination record with all calculated amounts, status DRAFT
      9. Return termination with calculation breakdown
    - `confirmTermination(terminationId, ctx)`:
      - Transition status DRAFT → PROCESSED
      - Transition employee.status to DESLIGADO
      - Set employee.terminationDate
      - Set processedAt = now()
      - Generate TRCT PDF and store URL
    - `markAsPaid(terminationId, ctx)`:
      - Transition PROCESSED → PAID
    - `listTerminations(orgId, filters, ctx)`: filter by terminationType, status, date range
    - `getTerminationById(id, orgId, ctx)`: single termination + employee data
    - `getTrctPdf(terminationId, orgId, ctx)`: return TRCT PDF buffer
    - `getGrfPdf(terminationId, orgId, ctx)`: return GRRF PDF buffer
    - `getExpiringDeadlines(orgId, daysAhead)`: terminations where paymentDeadline - daysAhead <= today
