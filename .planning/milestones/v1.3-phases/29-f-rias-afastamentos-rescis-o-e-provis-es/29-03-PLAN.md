    **payroll-provisions.service.ts:**
    Pure calculation function (no DB):
    - `calculateMonthlyProvision(salary: Decimal, ratPercent: Decimal): ProvisionCalcResult`:
      - vacationProvision = salary.div(12).mul(new Decimal('1.333333')).toDecimalPlaces(2, Decimal.ROUND_HALF_UP)
      - thirteenthProvision = salary.div(12).toDecimalPlaces(2, Decimal.ROUND_HALF_UP)
      - chargeRate = new Decimal('0.20').add(ratPercent).add(new Decimal('0.08'))
      - vacationCharges = vacationProvision.mul(chargeRate).toDecimalPlaces(2)
      - thirteenthCharges = thirteenthProvision.mul(chargeRate).toDecimalPlaces(2)

    Service functions (with DB):
    - `calculateMonthlyProvisions(input: CalculateProvisionsInput, ctx)`:
      1. Check if provisions already exist for this month → 409 if so
      2. Load all ACTIVE employees for the organization
      3. Load EngineParams (for ratPercent)
      4. For each employee (per-employee transaction, NOT one big tx — per Phase 28 decision):
         a. Get latest salary from EmployeeSalaryHistory
         b. Get cost center from Timesheet for referenceMonth (if exists)
         c. Call calculateMonthlyProvision(salary, ratPercent)
         d. Create 2 PayrollProvision records (VACATION + THIRTEENTH) with:
            - accountingEntryJson: { debitAccount: "6.1.01"/"6.1.02", debitLabel: "Despesa com Ferias"/"Despesa com 13o", creditAccount: "2.2.01"/"2.2.02", creditLabel: "Provisao de Ferias a Pagar"/"Provisao de 13o a Pagar", amount, referenceMonth, employeeId }
      5. Return summary: { processedCount, totalVacation, totalThirteenth, totalCharges }

    - `reverseProvision(provisionId, reversedBy, ctx)`:
      - Set reversedAt = now(), reversedBy
      - Do NOT delete the record (audit trail)

    - `listProvisions(orgId, referenceMonth, ctx)`: filter by month, group by employee
    - `getProvisionReport(orgId, referenceMonth, ctx)`:
      - Aggregate provisions by costCenterId and provisionType
      - Return summary rows with totals
    - `exportProvisionReport(orgId, referenceMonth, ctx)`: CSV export of report
