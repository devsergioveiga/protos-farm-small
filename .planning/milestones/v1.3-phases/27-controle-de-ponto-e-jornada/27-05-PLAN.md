**time-punch-repository.ts** — SQLite CRUD for offline punches:
```typescript
import * as SQLite from 'expo-sqlite';
import uuid from 'react-native-uuid';

export type PunchType = 'CLOCK_IN' | 'BREAK_START' | 'BREAK_END' | 'CLOCK_OUT';

export interface LocalTimePunch {
  id: string;
  employeeId: string;
  organizationId: string;
  farmId: string;
  punchType: PunchType;
  punchedAt: string;  // ISO 8601
  latitude: number | null;
  longitude: number | null;
  outOfRange: boolean;
  synced: boolean;
  createdAt: string;
}

export class TimePunchRepository {
  constructor(private db: SQLite.SQLiteDatabase) {}

  async create(punch: Omit<LocalTimePunch, 'id' | 'synced' | 'createdAt'>): Promise<LocalTimePunch> {
    // NOTE: organizationId is stored so offline-queue can build the correct endpoint URL
    const id = uuid.v4() as string;
    const createdAt = new Date().toISOString();
    await this.db.runAsync(
      `INSERT INTO time_punches (id, employee_id, organization_id, farm_id, punch_type, punched_at, latitude, longitude, out_of_range, synced, created_at)
       VALUES (?, ?, ?, ?, ?, ?, ?, ?, 0, ?)`,
      [id, punch.employeeId, punch.organizationId, punch.farmId, punch.punchType, punch.punchedAt, punch.latitude, punch.longitude, punch.outOfRange ? 1 : 0, createdAt]
    );
    return { ...punch, id, synced: false, createdAt };
  }

  async getTodayPunches(employeeId: string): Promise<LocalTimePunch[]> {
    const today = new Date().toISOString().split('T')[0]; // YYYY-MM-DD
    const rows = await this.db.getAllAsync(
      `SELECT * FROM time_punches WHERE employee_id = ? AND punched_at LIKE ? ORDER BY punched_at ASC`,
      [employeeId, `${today}%`]
    );
    return rows.map(this.mapRow);
  }

  async markSynced(id: string): Promise<void> {
    await this.db.runAsync(`UPDATE time_punches SET synced = 1 WHERE id = ?`, [id]);
  }

  async getPendingCount(): Promise<number> {
    const result = await this.db.getFirstAsync<{ count: number }>(
      `SELECT COUNT(*) as count FROM time_punches WHERE synced = 0`
    );
    return result?.count ?? 0;
  }

  private mapRow(row: any): LocalTimePunch {
    return {
      id: row.id,
      employeeId: row.employee_id,
      organizationId: row.organization_id,
      farmId: row.farm_id,
      punchType: row.punch_type,
      punchedAt: row.punched_at,
      latitude: row.latitude,
      longitude: row.longitude,
      outOfRange: row.out_of_range === 1,
      synced: row.synced === 1,
      createdAt: row.created_at,
    };
  }
}
```

**Implementation details:**

1. **GPS capture:** Use `Location.requestForegroundPermissionsAsync()` then `Location.getCurrentPositionAsync({ accuracy: Location.Accuracy.High })`. Max 3 second timeout for GPS. If denied or timeout: register punch without coordinates, set `outOfRange = false`, `noBoundary = false` (just missing GPS — not a geofence issue).

2. **Geofence validation (offline):** Use `@turf/boolean-within` to check if GPS point is inside farm boundary polygon. Farm boundary fetched and cached from `/api/org/${orgId}/farms/${farmId}` on screen mount (cache in AsyncStorage). If offline and no cached boundary: set `noBoundary = true`. If outside boundary: show warning modal "Voce esta fora do perimetro da fazenda. Registrar mesmo assim?" — yes/no. If yes: set `outOfRange = true`.

3. **Haptic feedback:**
   - On button press: `Haptics.impactAsync(Haptics.ImpactFeedbackStyle.Light)`
   - On confirmed entry: `Haptics.notificationAsync(Haptics.NotificationFeedbackType.Success)`

4. **State machine for button:**
   ```typescript
   type PunchState = 'IDLE' | 'CLOCKED_IN' | 'ON_BREAK';
   // Derive from today's punches:
   // No punches or last is CLOCK_OUT -> IDLE
   // Last is CLOCK_IN or BREAK_END -> CLOCKED_IN
   // Last is BREAK_START -> ON_BREAK
   ```

5. **Offline storage:** Every punch goes through TimePunchRepository.create() first (optimistic local save). Then enqueue in offline-queue. Show sync status dots.

6. **Offline banner:** Detect connectivity using `@react-native-community/netinfo` (already installed pattern — check mobile package.json). If offline: show banner with `WifiOff` icon + pending count.

7. **Elapsed time counter:** `useEffect` with `setInterval(1000)` while `punchState === 'CLOCKED_IN'`. Display as `HH:MM:SS` in JetBrains Mono.

8. **Clock-in button pulse animation:** When `punchState === 'CLOCKED_IN'`, ring pulse on `--color-primary-200` at 1500ms infinite. Suppress with `useReducedMotion()` from `react-native-reanimated` or manual `AccessibilityInfo.isReduceMotionEnabled()`.

9. **Accessibility:**
   - Clock-in/out button: `accessible={true}`, `accessibilityLabel="Registrar entrada"` / `"Registrar saida"` (dynamic), `accessibilityRole="button"`
   - GPS status chip: `accessibilityRole="text"`, `accessibilityLabel="Status: Dentro da fazenda"`
   - Offline banner: `accessibilityRole="alert"`
