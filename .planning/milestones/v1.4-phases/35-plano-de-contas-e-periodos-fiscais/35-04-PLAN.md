<interfaces>
Backend API (from Plans 02 and 03):

GET /api/org/:orgId/chart-of-accounts -> ChartOfAccountNode[]
  { id, code, name, parentId, level, accountType, nature, isSynthetic, allowManualEntry, isActive, isFairValueAdj, spedRefCode }

POST /api/org/:orgId/chart-of-accounts -> ChartOfAccountNode
  body: { code, name, parentId?, accountType, nature, isSynthetic?, allowManualEntry?, isFairValueAdj?, spedRefCode? }

PUT /api/org/:orgId/chart-of-accounts/:id -> ChartOfAccountNode
  body: Partial of create input + { isActive? }

DELETE /api/org/:orgId/chart-of-accounts/:id -> { message }

POST /api/org/:orgId/chart-of-accounts/seed -> { created, updated }

GET /api/org/:orgId/chart-of-accounts/unmapped-sped -> ChartOfAccountNode[]

GET /api/org/:orgId/fiscal-years -> FiscalYearOutput[]
  { id, name, startDate, endDate, isActive, periods: AccountingPeriodOutput[] }

POST /api/org/:orgId/fiscal-years -> FiscalYearOutput
  body: { name, startDate, endDate }

POST /api/org/:orgId/accounting-periods/:periodId/close -> AccountingPeriodOutput
  body: { closedBy }

POST /api/org/:orgId/accounting-periods/:periodId/reopen -> AccountingPeriodOutput
  body: { reopenedBy, reopenReason }

POST /api/org/:orgId/accounting-periods/:periodId/block -> AccountingPeriodOutput

Enums: AccountType (ATIVO|PASSIVO|PL|RECEITA|DESPESA), AccountNature (DEVEDORA|CREDORA), PeriodStatus (OPEN|CLOSED|BLOCKED)
