    Function `getAccountingDashboard(organizationId: string, filters: AccountingDashboardFilters): Promise<AccountingDashboardOutput>`:

    **KPI Cards** (per D-05):
    1. Call `getDre(organizationId, filters)` to get current DRE
    2. Call `getDre(organizationId, { fiscalYearId: filters.fiscalYearId, month: filters.month - 1 })` (prior month) — wrap in try/catch, default to '0' if month is 1 or fails
    3. Build 4 KPI cards:
       - "Resultado Acumulado": `dre.resultadoLiquido.ytd`; delta vs prior month's YTD
       - "Receita Total": sum receita sections ytd; delta vs prior
       - "Despesa Total": sum despesa sections ytd; delta vs prior
       - "Margem Operacional": extract from DRE sections the 'resultado-antes-ir' section if exists, else compute (receita - despesa) / receita * 100; delta vs prior
    4. Delta calculation: `((current - prior) / abs(prior)) * 100` — if prior is 0 return null

    **12-Month Revenue vs Expense Chart** (per D-05, Pattern 7 from RESEARCH):
    Use a single raw SQL query (NOT 12 calls to getDre):
    ```sql
    SELECT ab.month,
      SUM(CASE WHEN coa."accountType" = 'RECEITA' THEN ab."creditTotal" - ab."debitTotal" ELSE 0 END) as receita,
      SUM(CASE WHEN coa."accountType" = 'DESPESA' THEN ab."debitTotal" - ab."creditTotal" ELSE 0 END) as despesa
    FROM account_balances ab
    JOIN chart_of_accounts coa ON coa.id = ab."accountId"
    WHERE ab."organizationId" = $1
      AND ab."fiscalYearId" = $2
      AND coa."accountType" IN ('RECEITA', 'DESPESA')
      AND coa."isSynthetic" = false
    GROUP BY ab.month
    ORDER BY ab.month
    ```
    Use `prisma.$queryRaw` (same pattern as `computeMarginRanking` in financial-statements.service.ts).

    **Cost Composition Donut** (per D-05):
    From the same AccountBalance data (or DRE sections), group despesas by nature:
    - "CPV" (code prefix '5.1.')
    - "Despesas Administrativas" (prefix '5.2.01')
    - "Despesas Comerciais" (prefix '5.2.02')
    - "Despesas com Pessoal" (prefix '5.2.04' or '6.1.')
    - "Depreciacao" (prefix '5.2.03')
    - "Despesas Financeiras" (prefix '5.2.05' or '5.3.')
    Use raw SQL grouping by code prefix for YTD period. Calculate each as % of total.

    **Step 3: Create `accounting-dashboard.routes.ts`**:
    ```typescript
    import { Router } from 'express';
    import type { Request, Response } from 'express';
    import { authenticate } from '../../middleware/auth';
    import { checkPermission } from '../../middleware/check-permission';
    import { getAccountingDashboard } from './accounting-dashboard.service';
    import { FinancialStatementsError } from './financial-statements.types';

    export const accountingDashboardRouter = Router();

    accountingDashboardRouter.get(
      '/org/:orgId/accounting-dashboard',
      authenticate,
      checkPermission('financial:read'),
      async (req: Request, res: Response): Promise<void> => {
        try {
          const orgId = req.params.orgId as string;
          const fiscalYearId = req.query.fiscalYearId as string | undefined;
          const monthStr = req.query.month as string | undefined;

          if (!fiscalYearId) {
            res.status(400).json({ error: 'fiscalYearId e obrigatorio', code: 'MISSING_FISCAL_YEAR_ID' });
            return;
          }
          if (!monthStr) {
            res.status(400).json({ error: 'month e obrigatorio', code: 'MISSING_MONTH' });
            return;
          }
          const month = parseInt(monthStr, 10);
          if (isNaN(month) || month < 1 || month > 12) {
            res.status(400).json({ error: 'month deve ser entre 1 e 12', code: 'INVALID_MONTH' });
            return;
          }

          const result = await getAccountingDashboard(orgId, { fiscalYearId, month });
          res.json(result);
        } catch (err) {
          if (err instanceof FinancialStatementsError) {
            res.status(err.statusCode).json({ error: err.message, code: err.code });
            return;
          }
          if (err instanceof Error) {
            res.status(500).json({ error: err.message });
            return;
          }
          res.status(500).json({ error: 'Erro interno do servidor' });
        }
      },
    );
    ```
