       - Import `Money` from `@protos-farm/shared`, `FEBRABAN_BANK_MAP` from `@protos-farm/shared`
       - `createBankAccount(ctx, input)`: validate bankCode against FEBRABAN_BANK_MAP, use `withRlsContext` transaction to: (a) create BankAccount, (b) createMany BankAccountFarm if farmIds provided, (c) create BankAccountBalance with initialBalance/currentBalance set to input.initialBalance, (d) if initialBalance > 0 create FinancialTransaction with type CREDIT, referenceType OPENING_BALANCE.        - `updateBankAccount(ctx, id, input)`: update fields, replace farmIds (delete existing + createMany new) if provided.
       - `deleteBankAccount(ctx, id)`: set isActive=false (soft delete).
       - `getStatement(ctx, accountId, query)`: list FinancialTransactions for account, filter by transactionDate range (from/to) and type. Return sorted by transactionDate desc. Use Money.fromPrismaDecimal for amounts.
       - `getDashboard(ctx)`: aggregate all active accounts' currentBalance, group by type. Use Money for summing. Return totals.
