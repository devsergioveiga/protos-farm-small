    2. Add the following models to `apps/backend/prisma/schema.prisma` (append at end, before any closing comments):

       **enum BankAccountType:**
       - CHECKING, SAVINGS, INVESTMENT, RURAL_CREDIT

       **model BankAccount:**
       - id: String @id @default(uuid())
       - organizationId: String (FK to Organization, onDelete: Cascade)
       - name: String (display name like "Conta principal BB")
       - type: BankAccountType
       - bankCode: String (FEBRABAN code e.g. "001")
       - agency: String
       - agencyDigit: String? (optional)
       - accountNumber: String
       - accountDigit: String? (optional)
       - producerId: String? (OPTIONAL FK to Producer — per user decision, null for org-level accounts)
       - notes: String?
       - isActive: Boolean @default(true)
       - createdAt: DateTime @default(now())
       - updatedAt: DateTime @updatedAt
       - Relations: organization, producer?, farms (BankAccountFarm[]), balance (BankAccountBalance?), transactions (FinancialTransaction[])
       - Indexes: [organizationId], [organizationId, type]
       - @@map("bank_accounts")

       **model BankAccountFarm:** (N:N junction table per user decision)
       - id: String @id @default(uuid())
       - bankAccountId: String (FK to BankAccount, onDelete: Cascade)
       - farmId: String (FK to Farm, onDelete: Cascade)
       - @@unique([bankAccountId, farmId])
       - @@map("bank_account_farms")

       **model BankAccountBalance:** (aggregate pattern mirroring StockBalance)
       - id: String @id @default(uuid())
       - bankAccountId: String @unique (FK to BankAccount, onDelete: Cascade)
       - organizationId: String (FK to Organization)
       - initialBalance: Decimal @default(0) @db.Decimal(15, 2)
       - currentBalance: Decimal @default(0) @db.Decimal(15, 2)
       - updatedAt: DateTime @updatedAt
       - @@index([organizationId])
       - @@map("bank_account_balances")

       **model FinancialTransaction:** (ledger for extrato)
       - id: String @id @default(uuid())
       - organizationId: String (FK to Organization)
       - bankAccountId: String (FK to BankAccount)
       - type: String ("CREDIT" | "DEBIT")
       - amount: Decimal @db.Decimal(15, 2)
       - description: String
       - referenceType: String? ("OPENING_BALANCE" | "PAYABLE" | "RECEIVABLE" | "TRANSFER")
       - referenceId: String?
       - transactionDate: DateTime
       - createdAt: DateTime @default(now())
       - @@index([organizationId, bankAccountId])
       - @@index([organizationId, bankAccountId, transactionDate])
       - @@map("financial_transactions")

    3. Add reverse relations on existing models:
       - Organization: add `bankAccounts BankAccount[]`, `bankAccountBalances BankAccountBalance[]`, `financialTransactions FinancialTransaction[]`
       - Producer: add `bankAccounts BankAccount[]`
       - Farm: add `bankAccountFarms BankAccountFarm[]`

    4. Run migration: `cd apps/backend && npx prisma migrate dev --name add_bank_accounts`

    5. Run generate: `cd apps/backend && npx prisma generate`
