Expected CSV/XLSX columns: `Tipo,Nome/Razão Social,Nome Fantasia,CNPJ/CPF,IE,Endereço,Cidade,UF,CEP,Contato Nome,Contato Telefone,Contato Email,Condição Pagamento,Frete,Categorias`

Validation per row:

- `Tipo` must be "PF" or "PJ" (case-insensitive). If empty, infer from document length (11=PF, 14=PJ after cleaning).
- `Nome/Razão Social` required, min 2 chars.
- `CNPJ/CPF` required, validated via `isValidCNPJ`/`isValidCPF` from document-validator.ts. Store cleaned (digits only).
- `UF` if present must be in VALID_UF list.
- `Categorias` parsed as pipe-separated (`|`), each must match a `SupplierCategory` enum value (case-insensitive mapping: "Insumo Agrícola" -> "INSUMO_AGRICOLA", "Pecuário" -> "PECUARIO", etc.). If empty, defaults to `['OUTROS']`.
- `Frete` if present must be "CIF" or "FOB" (case-insensitive).

For XLSX: use ExcelJS to read first sheet, row 1 as headers. For CSV: split by `;` (semicolon, pt-BR standard) or `,` (detect from first line).

```typescript
import multer from 'multer';
import { parseSupplierFile } from './supplier-file-parser';

const upload = multer({
  storage: multer.memoryStorage(),
  limits: { fileSize: 5 * 1024 * 1024 },
  fileFilter: (_req, file, cb) => {
    const allowed = ['text/csv', 'application/vnd.openxmlformats-officedocument.spreadsheetml.sheet', 'application/vnd.ms-excel'];
    if (allowed.includes(file.mimetype) || file.originalname.endsWith('.csv') || file.originalname.endsWith('.xlsx')) {
      cb(null, true);
    } else {
      cb(new Error('Formato não suportado. Use CSV (.csv) ou Excel (.xlsx).'));
    }
  },
});

suppliersRouter.post(`${base}/import/preview`, checkPermission('purchases:manage'), upload.single('file'), ...);
suppliersRouter.post(`${base}/import/execute`, checkPermission('purchases:manage'), upload.single('file'), ...);
suppliersRouter.get(`${base}/import/template`, checkPermission('purchases:read'), ...);
```

  - Format rating as `averageRating.toFixed(1)` or empty string if no ratings.
  - Return string.

- `exportSuppliersPdf(ctx, query: ListSuppliersQuery, orgName: string)`:
  - Fetch all matching suppliers (same as CSV).

  - Use alternating row backgrounds (light gray every other row).
  - Return Buffer (collect chunks in `data` event, resolve in `end` event).

- `createRating(ctx, supplierId: string, input: CreateRatingInput, ratedBy: string)`:
  - Validate supplier exists (not deleted). Throw 404 if not found.
  - Validate all 4 criteria are integers 1-5. Throw 400 if invalid.
  - `tx.supplierRating.create({ data: { supplierId, organizationId: ctx.organizationId, ...input, ratedBy } })`.
  - Return created rating.

- `listRatings(ctx, supplierId: string)`:
  - Validate supplier exists. Throw 404 if not.
  - `tx.supplierRating.findMany({ where: { supplierId }, orderBy: { createdAt: 'desc' } })`.
  - For each rating, compute individual average: `(deadline + quality + price + service) / 4`.
  - Return array.
