**Service (`tax-guides.service.ts`):**

Create class `TaxGuidesService` with methods:

1. `generateGuides(orgId, input: GenerateTaxGuidesInput, userId)`:
   - Within `withRlsContext` transaction:
     a. Query `PayrollRunItem` aggregated by referenceMonth matching input, grouped to sum `fgtsAmount`, `inssAmount`, `irrfAmount`, `grossSalary` (for FUNRURAL on gross revenue basis)
     b. For FUNRURAL: look up `Organization.funruralBasis`. If 'PAYROLL', FUNRURAL amount = sum of grossSalary * 2.7%. If 'GROSS_REVENUE', look up `PayrollLegalTable` where `tableType='FUNRURAL'` and `effectiveFrom <= referenceMonth ORDER BY effectiveFrom DESC LIMIT 1`, extract rate from brackets JSON, multiply by organization's gross revenue (for now, use payroll gross as proxy since revenue tracking is Phase 32)
     c. Calculate dueDate for each type using `TAX_GUIDE_DUE_DAYS`: add to the 1st of the NEXT month. If dueDate falls on Saturday, move to Monday; if Sunday, move to Monday.
     d. Upsert `TaxGuide` for each type (unique constraint on orgId+guideType+referenceMonth handles idempotency)
     e. For each guide, `tx.payable.upsert` with `originType: 'TAX_GUIDE'`, `originId: taxGuide.id`, `category: 'TAXES' as const`, `supplierName` = guide type label (e.g. 'FGTS - Competencia 03/2026'), `dueDate` from step c, `totalAmount` from step b
   - Return array of TaxGuideOutput

2. `listGuides(orgId, query: ListTaxGuidesQuery)`:
   - Query with filters, compute `daysUntilDue` and `alertLevel` for each: `alertLevel = daysUntilDue <= 5 ? 'danger' : daysUntilDue <= 10 ? 'warning' : 'none'`
   - Paginated response

3. `downloadGuide(orgId, guideId)`:
   - Fetch guide, generate file based on guideType:
     - FGTS: call `buildSefipRE(guide, payrollItems)` — fixed-width text file per SEFIP 8.4 spec
     - INSS/IRRF: call `buildDarfPdf(guide, org)` — pdfkit PDF replicating DARF form layout
     - FUNRURAL: call `buildGpsPdf(guide, org)` — pdfkit PDF replicating GPS form layout
   - Update guide status to GENERATED, set generatedAt
   - Return { buffer, filename, contentType }

4. Private `buildSefipRE(guide, items)`:
   - Generate fixed-width .RE file with records: 10 (header), 20 (empregador with FPAS=604), 30 (trabalhador per employee), 32 (remuneracao), 40 (totais trabalhador), 50 (totais empregador), 99 (trailer)
   - Each record type has fixed column positions per CAIXA SEFIP 8.4 manual
   - Competencia format: AAAAMM (e.g. 202603)
   - Character encoding: ASCII/Latin-1

5. Private `buildDarfPdf(guide, org)`:
   - Use pdfkit to create A4 PDF with DARF layout
   - Fields: org CNPJ, Receita code from TAX_GUIDE_RECEITA_CODES, competencia, due date, valor principal = totalAmount, valor total = totalAmount
   - Include DARF numeric code (long string for internet banking payment)

6. Private `buildGpsPdf(guide, org)`:
   - Similar to DARF but GPS layout
   - Codigo pagamento: 2100, competencia, INSS value, outras entidades (SENAR portion)

**Hook (`useTaxGuides.ts`):**
Following the existing useState+useCallback pattern (matching usePayrollRuns):

```typescript
export function useTaxGuides() {
  const [guides, setGuides] = useState<TaxGuide[]>([]);
  const [loading, setLoading] = useState(false);
  const [error, setError] = useState<string | null>(null);
  const [successMessage, setSuccessMessage] = useState<string | null>(null);

  const fetchGuides = useCallback(async (params?: { referenceMonth?: string; guideType?: string; status?: string }) => { ... }, []);
  const generateGuides = useCallback(async (input: GenerateTaxGuidesInput) => { ... }, []);
  const downloadGuide = useCallback(async (id: string, guideType: string) => {
    // Use api.getBlob for file download, open in new tab
    const blob = await api.getBlob(`/org/tax-guides/${id}/download`);
    const url = URL.createObjectURL(blob);
    const ext = guideType === 'FGTS' ? '.RE' : '.pdf';
    // Create temp link and trigger download
    const a = document.createElement('a');
    a.href = url;
    a.download = `guia-${guideType}-${Date.now()}${ext}`;
    a.click();
    setTimeout(() => URL.revokeObjectURL(url), 10000);
  }, []);

  return { guides, loading, error, successMessage, fetchGuides, generateGuides, downloadGuide };
}
```
