**2. Create `hooks/useQuotations.ts`:**
Follow usePurchaseRequests.ts pattern exactly:
- `useQuotations(filters)` — fetches from `/org/quotations` with pagination, returns { quotations, isLoading, error, total, totalPages, refetch }
- `useQuotation(id)` — fetches single from `/org/quotations/${id}`, returns { quotation, isLoading, error, refetch }
- `useComparativeMap(quotationId)` — fetches from `/org/quotations/${id}/comparative`, returns { data: ComparativeMapData | null, isLoading, error, refetch }
- `createQuotation(input)` — POST to `/org/quotations`
- `registerProposal(quotationId, qsId, data, file?)` — POST multipart to `/org/quotations/${id}/suppliers/${qsId}/proposal`
- `approveQuotation(quotationId, input)` — PATCH to `/org/quotations/${id}/approve`
- `transitionQuotation(quotationId, status)` — PATCH to `/org/quotations/${id}/transition`
- `deleteQuotation(quotationId)` — DELETE to `/org/quotations/${id}`
