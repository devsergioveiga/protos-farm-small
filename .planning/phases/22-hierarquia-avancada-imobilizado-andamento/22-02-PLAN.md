- `createRenovation(ctx: RlsContext, assetId: string, input: CreateRenovationInput)`:
  - Use `prisma.$transaction` (direct, NOT withRlsContext — STATE.md decision).
  - Fetch asset: `tx.asset.findFirst({ where: { id: assetId, organizationId: ctx.organizationId, deletedAt: null } })`.
  - Guard: if `!asset` throw 404 "Ativo nao encontrado".
  - Guard: if `asset.status === 'ALIENADO'` throw 400 "Ativo alienado nao pode ser reformado".
  - Guard: if `asset.status === 'EM_ANDAMENTO'` throw 400 "Ativo em andamento nao pode ser reformado".
  - Create renovation record: `tx.assetRenovation.create({ data: { organizationId: ctx.organizationId, assetId, description: input.description, renovationDate: new Date(input.renovationDate), totalCost: new Decimal(String(input.totalCost)), accountingDecision: input.accountingDecision, newUsefulLifeMonths: input.newUsefulLifeMonths ?? null, notes: input.notes ?? null } })`.
  - If `input.accountingDecision === 'CAPITALIZAR'`:
    - `tx.asset.update({ where: { id: assetId }, data: { acquisitionValue: { increment: new Decimal(String(input.totalCost)).toNumber() } } })`.
    - If `input.newUsefulLifeMonths`:
      - `tx.depreciationConfig.updateMany({ where: { assetId, organizationId: ctx.organizationId }, data: { usefulLifeMonths: input.newUsefulLifeMonths } })`.
  - Return renovation record.
- `listRenovations(ctx: RlsContext, assetId: string)`:
  - `prisma.assetRenovation.findMany({ where: { assetId, organizationId: ctx.organizationId }, orderBy: { renovationDate: 'desc' } })`.
  - Return list.

2. Create `asset-wip.service.ts`:

- `addContribution(ctx, assetId, input)`:
  - `prisma.$transaction(async (tx) => { ... })` (direct, NOT withRlsContext).
  - Fetch asset: must have `status === 'EM_ANDAMENTO'` and `deletedAt === null`. If not, throw 400 "Apenas ativos em andamento aceitam aportes".
  - Create contribution: `tx.assetWipContribution.create({ data: { organizationId: ctx.organizationId, assetId, contributionDate: new Date(input.contributionDate), amount: new Decimal(String(input.amount)), description: input.description, stageId: input.stageId ?? null, supplierId: input.supplierId ?? null, invoiceRef: input.invoiceRef ?? null, notes: input.notes ?? null } })`.
  - Aggregate total: `tx.assetWipContribution.aggregate({ where: { assetId, organizationId: ctx.organizationId }, _sum: { amount: true } })`.
  - Compute budget flags:
    ```typescript
    const totalContributed = Number(agg._sum.amount ?? 0);
    const budget = asset.wipBudget ? Number(asset.wipBudget) : null;
    const alertPct = asset.wipBudgetAlertPct ? Number(asset.wipBudgetAlertPct) : 90;
    const budgetAlert = budget !== null && totalContributed / budget >= alertPct / 100;
    const budgetExceeded = budget !== null && totalContributed > budget;
    ```
  - Return `{ contribution, totalContributed, budgetAlert, budgetExceeded }`.
- `getWipSummary(ctx, assetId)`:
  - Fetch asset with wipBudget, wipBudgetAlertPct, name, assetTag, status.
  - Guard: `status !== 'EM_ANDAMENTO'` -> throw 400.
  - Aggregate totalContributed.
  - Count contributions.
  - Fetch stages: `tx.assetWipStage.findMany({ where: { assetId, organizationId }, orderBy: { sortOrder: 'asc' } })`.
  - Fetch contributions: `tx.assetWipContribution.findMany({ where: { assetId, organizationId }, orderBy: { contributionDate: 'desc' } })`.
  - Compute budget flags.
  - Return `{ assetId, assetName, assetTag, status, budget, budgetAlertPct, totalContributed, contributionCount, budgetAlert, budgetExceeded, stages, contributions }`.
- `activateWipAsset(ctx, assetId, input)`:
  - `prisma.$transaction(async (tx) => { ... })`.
  - Fetch asset: must be `status === 'EM_ANDAMENTO'`. If not, throw 404 "Ativo em andamento nao encontrado".
  - Aggregate totalContributed (same as above).
  - Update asset: `tx.asset.update({ where: { id: assetId }, data: { status: 'ATIVO', acquisitionDate: input.activationDate ? new Date(input.activationDate) : new Date(), acquisitionValue: totalContributed } })`.
  - Check DepreciationConfig exists: `tx.depreciationConfig.findFirst({ where: { assetId, organizationId } })`.
  - Return `{ assetId, finalValue: totalContributed, depreciationConfigMissing: !depConfig }`.
