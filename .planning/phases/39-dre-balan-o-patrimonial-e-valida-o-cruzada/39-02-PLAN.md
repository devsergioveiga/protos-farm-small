**DreTable.tsx** — Semantic `<table>` component per UI-SPEC:

Props: `{ sections: DreSection[]; resultadoLiquido: DreSectionRow; showVH: boolean; loading: boolean; period: string }`

- `<caption>`: "Demonstracao do Resultado do Exercicio — {period}"
- `<thead>`: columns — "Descricao", "Mes Atual", "Acumulado", "Mesmo Per. Ano Ant." (always), plus "% AV" and "Delta% AH" when showVH=true
- `<tbody>`: iterate sections, for each section:
  - Section header row: `<tr>` with `className="dre-table__section-header"` (Source Sans 3 14px 600, bg neutral-50, padding-left 16px). `<th scope="row">`.
  - Account rows: `<tr>` with `className="dre-table__account-row"` (14px 400, white bg, padding-left 32px). CPC 29 rows get additional `dre-table__cpc29` class (background `--color-sun-100` #FFF9C4).
  - Subtotal rows: `<tr>` with `className="dre-table__subtotal"` (14px 600, bg neutral-100, top border 1px neutral-200).
- `<tfoot>`: Resultado Liquido row (bold, top border 2px).
- All monetary values formatted with `Intl.NumberFormat('pt-BR', { style: 'currency', currency: 'BRL' })`.
- AV% column: `Intl.NumberFormat('pt-BR', { style: 'percent', minimumFractionDigits: 1 })`.
- AH% column: green (`--color-success-500`) for positive, red (`--color-error-500`) for negative, neutral for 0%.
- V/H columns: use `opacity: 0` → `opacity: 1` transition 200ms (per animation contract). Columns pre-allocated with `col-width` so no layout shift.
- Loading state: 12 skeleton `<tr>` rows with alternating indent, pulse animation `opacity 0.4 → 0.7` 1500ms.
- Accessibility: `<th scope="col">` for column headers, `<th scope="row">` for row labels.
