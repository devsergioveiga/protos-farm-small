**Create `asset-reports.service.ts`:**

Three exported functions:

1. `getInventoryReport(query: InventoryReportQuery): Promise<InventoryReportResult>` — Two-query pattern:
   - Query 1: `prisma.asset.groupBy({ by: ['classification'], where: { organizationId, deletedAt: null, status: { not: 'ALIENADO' } }, _sum: { acquisitionValue: true }, _count: true })`. Apply optional farmId filter.
   - Query 2: Get all assetIds with their classification, then `prisma.depreciationEntry.groupBy({ by: ['assetId'], where: { organizationId, reversedAt: null }, _sum: { depreciationAmount: true } })`.
   - In-memory: map assetId->classification, roll up depr sums by classification. Compute `netBookValue = grossValue - accumulatedDepreciation`.
   - If dateFrom/dateTo provided: count acquisitions where `acquisitionDate BETWEEN dateFrom AND dateTo`, and disposals from `prisma.assetDisposal.count(...)`.
   - Convert all Decimals with `.toNumber()` before returning.

2. `getDepreciationProjection(query: DepreciationProjectionQuery): Promise<DepreciationProjectionResult>` — Forward simulation:
   - Fetch all active assets (status not ALIENADO/EM_ANDAMENTO, deletedAt null) with their `depreciationConfig` relation and latest `depreciationEntry` (orderBy periodYear desc, periodMonth desc, take 1).
   - For each asset with a config: determine `openingBookValue` = latest entry's `closingBookValue` or asset's `acquisitionValue` if no entries.
   - For assets with HOURS_OF_USE or UNITS_OF_PRODUCTION method: fall back to STRAIGHT_LINE using `managerialAnnualRate` (or `fiscalAnnualRate`). Increment `assetsEstimated` counter.
   - Determine starting period: `nextPeriod(lastEntry.periodYear, lastEntry.periodMonth)` or current month if no entries.
   - Loop `horizonMonths` times: call `computeDepreciation()` from `../depreciation/depreciation-engine.service`, accumulate monthly totals.
   - Helper `nextPeriod(y, m)`: if m===12 return {year:y+1, month:1} else {year:y, month:m+1}.
   - Aggregate all assets' monthly amounts into cumulative series.

3. `getTCOFleet(query: TCOFleetQuery): Promise<TCOFleetResult>` — Fleet-level aggregation:
   - Fetch all assets (not deleted, not ALIENADO) with select: id, name, tag, assetType, acquisitionValue, currentHourmeter.
   - Aggregate depreciation per asset: `prisma.depreciationEntry.groupBy({ by: ['assetId'], where: { organizationId, reversedAt: null }, _sum: { depreciationAmount: true } })`.
   - Aggregate maintenance per asset: `prisma.workOrder.groupBy({ by: ['assetId'], where: { organizationId, status: 'CONCLUIDA' }, _sum: { totalCost: true } })`.
   - Aggregate fuel per asset: `prisma.fuelRecord.groupBy({ by: ['assetId'], where: { organizationId }, _sum: { totalCost: true } })`.
   - Build maps, compute per-asset TCO row. Repair ratio: `maintenanceCost / acquisitionValue`. Guard: if acquisitionValue is null or 0, set `repairRatio: null, alert: 'NO_DATA'`. Otherwise: >=0.70 -> 'REPLACE', >=0.60 -> 'MONITOR', else 'OK'.
   - Cost per hour: `totalCost / currentHourmeter` if hourmeter > 0.
   - Summary: average of costPerHour (excluding nulls), sum of maintenance, sum of fuel.

4. `exportInventoryReport(query: InventoryReportQuery, format: ExportFormat): Promise<Buffer>` — Delegates to getInventoryReport then:
   - PDF: Use pdfkit (dynamic import `(await import('pdfkit')).default`), landscape A4, margin 40. Header "Relatorio Patrimonial". Table with columns: Classificacao, Qtd, Valor Bruto, Depr. Acumulada, Valor Liquido. Totals row. Pattern: `asset-export.service.ts exportAssetsPdf()`.
   - XLSX: Use ExcelJS workbook. Sheet "Inventario Patrimonial". Columns: Classificacao(25), Qtd(12), Valor Bruto(20), Depr Acumulada(20), Valor Liquido(20). Bold header row. Pattern: `depreciation.service.ts exportReport()`.
   - CSV: Use ExcelJS workbook with `workbook.csv.writeBuffer()`.

All Decimal arithmetic must use `decimal.js`. Convert to Number only at return boundary.

**Create `asset-reports.routes.ts`:**

```typescript
import { Router } from 'express';
import { requireAuth, requirePermission } from '../../middleware/auth';
import * as service from './asset-reports.service';

export const assetReportsRouter = Router();

// All endpoints require auth + assets:read permission (consistent with asset module RBAC per STATE.md decision)
const base = '/orgs/:orgId/asset-reports';

// GET /orgs/:orgId/asset-reports/inventory
assetReportsRouter.get(`${base}/inventory`, requireAuth, requirePermission('assets:read'), async (req, res, next) => {
  try {
    const result = await service.getInventoryReport({
      organizationId: req.params.orgId,
      farmId: req.query.farmId as string | undefined,
      assetType: req.query.assetType as string | undefined,
      dateFrom: req.query.dateFrom as string | undefined,
      dateTo: req.query.dateTo as string | undefined,
    });
    res.json(result);
  } catch (err) { next(err); }
});

// GET /orgs/:orgId/asset-reports/inventory/export?format=pdf|xlsx|csv
assetReportsRouter.get(`${base}/inventory/export`, requireAuth, requirePermission('assets:read'), async (req, res, next) => {
  try {
    const format = (req.query.format as string) || 'pdf';
    if (!['pdf', 'xlsx', 'csv'].includes(format)) {
      return res.status(400).json({ error: 'Formato invalido. Use: pdf, xlsx ou csv' });
    }
    const buffer = await service.exportInventoryReport(
      { organizationId: req.params.orgId, farmId: req.query.farmId as string | undefined, assetType: req.query.assetType as string | undefined, dateFrom: req.query.dateFrom as string | undefined, dateTo: req.query.dateTo as string | undefined },
      format as 'pdf' | 'xlsx' | 'csv',
    );
    const contentTypes = { pdf: 'application/pdf', xlsx: 'application/vnd.openxmlformats-officedocument.spreadsheetml.sheet', csv: 'text/csv' };
    const extensions = { pdf: 'pdf', xlsx: 'xlsx', csv: 'csv' };
    res.setHeader('Content-Type', contentTypes[format as keyof typeof contentTypes]);
    res.setHeader('Content-Disposition', `attachment; filename="relatorio-patrimonial.${extensions[format as keyof typeof extensions]}"`);
    res.send(buffer);
  } catch (err) { next(err); }
});

// GET /orgs/:orgId/asset-reports/depreciation-projection?horizonMonths=12|36|60
assetReportsRouter.get(`${base}/depreciation-projection`, requireAuth, requirePermission('assets:read'), async (req, res, next) => {
  try {
    const horizonMonths = parseInt(req.query.horizonMonths as string, 10) || 12;
    if (![12, 36, 60].includes(horizonMonths)) {
      return res.status(400).json({ error: 'horizonMonths deve ser 12, 36 ou 60' });
    }
    const result = await service.getDepreciationProjection({
      organizationId: req.params.orgId,
      horizonMonths: horizonMonths as 12 | 36 | 60,
      farmId: req.query.farmId as string | undefined,
      assetType: req.query.assetType as string | undefined,
    });
    res.json(result);
  } catch (err) { next(err); }
});

// GET /orgs/:orgId/asset-reports/tco-fleet
assetReportsRouter.get(`${base}/tco-fleet`, requireAuth, requirePermission('assets:read'), async (req, res, next) => {
  try {
    const result = await service.getTCOFleet({
      organizationId: req.params.orgId,
      farmId: req.query.farmId as string | undefined,
      assetType: req.query.assetType as string | undefined,
    });
    res.json(result);
  } catch (err) { next(err); }
});
```
