```typescript
export interface NotificationPref {
  eventType: string;
  channel: string;
  enabled: boolean;
}

interface UseNotificationPreferencesResult {
  preferences: NotificationPref[];
  isLoading: boolean;
  updatePreference: (eventType: string, channel: string, enabled: boolean) => void;
  savePreferences: () => Promise<void>;
  isDirty: boolean;
}
```

- Fetch on mount: `api.get<NotificationPref[]>('/org/notifications/preferences')`
- Build a local map of all event types x channels with default `enabled: true`
- Merge fetched preferences over defaults (overwrite enabled flag)
- `updatePreference`: update local state (mark dirty)
- `savePreferences`: PATCH `/org/notifications/preferences` with `{ preferences: changedPrefs }`
