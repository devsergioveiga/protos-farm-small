2. **Create `purchasing-dashboard.service.ts`:**

Export `getDashboardData(ctx: RlsContext, filters: PurchasingDashboardFilters): Promise<PurchasingDashboardData>`.

Implementation using `withRlsContext`:

- **KPIs**: Execute parallel queries for current and previous period:
  - `pendingApprovalCount`: Count PurchaseRequest WHERE status IN ('PENDENTE', 'DEVOLVIDA') AND createdAt within period. Previous period = same duration shifted back.
  - `overduePoCount`: Count PurchaseOrder WHERE status IN ('EMITIDA', 'CONFIRMADA', 'EM_TRANSITO') AND expectedDeliveryDate < now. Previous = same logic in prior period.
  - `avgCycleDays`: Average of `(GoodsReceipt.confirmedAt - PurchaseRequest.createdAt)` in days for GRs confirmed within period. Use raw SQL or Prisma aggregation. If no GRs, return 0.
  - `lateDeliveriesCount`: Count GoodsReceipt WHERE confirmedAt > PurchaseOrder.expectedDeliveryDate within period.

- **Charts**:
  - `volumeByStage`: Count + sum of totalValue per kanban column status. Reuse column mapping logic from kanban service.
  - `purchasesByCategory`: Group PurchaseRequest items by product category, sum values. Top 8 + "Outros".
  - `monthlyEvolution`: Last 12 months of PurchaseOrder totalValue by month. Use `GROUP BY date_trunc('month', createdAt)`.
  - `urgentVsPlanned`: Last 6 months, PurchaseRequest count split by urgency='EMERGENCIAL'|'URGENTE' (urgent) vs 'NORMAL' (planned).

- **Alerts**: Same as KPI counts but always current (not period-filtered).

- Apply farmId filter to all queries if provided.

```typescript
import cron from 'node-cron';
import { redis } from '../../database/redis';
import { sendDigestEmails } from '../mail/digest-mail.service';

export function startDigestCron(): void {
  cron.schedule(
    '0 7 * * *',
    async () => {
      const lockKey = 'cron:daily-digest';
      const locked = await redis.set(lockKey, '1', 'EX', 120, 'NX');
      if (!locked) return;
      try {
        await sendDigestEmails();
      } finally {
        await redis.del(lockKey);
      }
    },
    { timezone: 'America/Sao_Paulo' },
  );
}
```
