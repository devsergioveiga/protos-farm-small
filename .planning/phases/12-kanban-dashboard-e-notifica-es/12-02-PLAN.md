**6. Create purchase-dashboard.service.ts:**

- `getDashboardMetrics(ctx: RlsContext, filters: DashboardFilters): Promise<{ metrics: DashboardMetrics; charts: DashboardCharts }>`:
  - Build comparison period using periodMs subtraction pattern (see interfaces)
  - Use `Promise.all` for parallel queries (NOT $transaction — read-only):
    - totalVolume: sum PurchaseOrder totals where createdAt in range, status NOT CANCELADA
    - requestCount: count PurchaseRequest where createdAt in range, status NOT IN (RASCUNHO, CANCELADA)
    - avgCycleTimeDays: find RCs that have completed cycle (Payable PAID) in range, compute avg days from RC.createdAt to Payable.paidAt
    - onTimeDeliveryPct: count POs where GoodsReceipt.receivedAt <= PO.expectedDeliveryDate / total POs with GR
    - accumulatedSaving: sum from QuotationItemSelection saving field (if exists) or compute from max price - selected price across quotation suppliers
  - Charts:
    - purchasesByCategory: group PO totals by RC requestType, normalize all categories (fill missing with 0)
    - savingEvolution: group saving by month for last 12 months
    - budgetVsActual: join PurchaseBudget (budget) with actual PO spend per category, normalize all categories
  - Run same queries for previous period to compute KpiValue.changePercent
  - Use in-memory cache with 5-min TTL: key = `${organizationId}:${startDate}:${endDate}:${farmId??'all'}:${category??'all'}`
  - Cache implementation: module-level `Map<string, { data: unknown; expiresAt: number }>`, check expiresAt on read, delete expired entries

- `getDashboardAlerts(ctx: RlsContext, filters: { farmId?: string }): Promise<DashboardAlert[]>`:
  - Pending RC aging: count RCs with status PENDENTE or EM_ANALISE older than 3 days
  - PO overdue: count POs where expectedDeliveryDate < now and status not ENTREGUE/CANCELADA
  - Budget overage: find categories where actual spend > budget amount from PurchaseBudget

Implement 5-min TTL cache as a module-level Map:

```typescript
const cache = new Map<string, { data: unknown; expiresAt: number }>();
const CACHE_TTL_MS = 5 * 60 * 1000;
function getCached<T>(key: string): T | null {
  const entry = cache.get(key);
  if (!entry) return null;
  if (Date.now() > entry.expiresAt) {
    cache.delete(key);
    return null;
  }
  return entry.data as T;
}
function setCache(key: string, data: unknown): void {
  cache.set(key, { data, expiresAt: Date.now() + CACHE_TTL_MS });
}
```

```typescript
// GET /org/notifications/preferences
notificationsRouter.get(`${base}/preferences`, authenticate, async (req, res) => {
  try {
    const ctx = buildRlsContext(req);
    const { getPreferences } = await import('./notification-preferences.service');
    const prefs = await getPreferences(ctx);

  } catch (err) {
    handleError(err, res);
  }
});

// PATCH /org/notifications/preferences
notificationsRouter.patch(`${base}/preferences`, authenticate, async (req, res) => {
  try {
    const ctx = buildRlsContext(req);
    const { preferences } = req.body as {
      preferences: { eventType: string; channel: string; enabled: boolean }[];
    };
    if (!Array.isArray(preferences)) {
      res.status(400).json({ error: 'Campo preferences deve ser um array' });
      return;
    }
    const { updatePreferences } = await import('./notification-preferences.service');
    await updatePreferences(ctx, preferences);
    res.json({ success: true });
  } catch (err) {
    handleError(err, res);
  }
});
```
