1. **Create `KanbanBoard.tsx`:**

```typescript
import {
  DndContext,
  DragOverlay,
  closestCenter,
  type DragStartEvent,
  type DragEndEvent,
  type DragOverEvent,
} from '@dnd-kit/core';
import { restrictToHorizontalAxis } from '@dnd-kit/modifiers';
```

Props: `columns: KanbanColumn[]`, `onCardMove: (cardId, from, to) => Promise<void>`.

State:

- `activeCard: KanbanCard | null` — currently dragged card
- `activeColumnId: KanbanColumnId | null` — source column of active card
- `confirmPending: { card, from, to } | null` — pending confirmation

Handlers:

- `onDragStart(event: DragStartEvent)`: Set activeCard from event.active.data.current.card. Set activeColumnId.
- `onDragOver(event: DragOverEvent)`: Compute valid/invalid targets from KANBAN_VALID_DROPS[activeColumnId]. Pass as props to KanbanColumn.
- `onDragEnd(event: DragEndEvent)`: If over is a valid column different from source, set `confirmPending = { card, from: activeColumnId, to: over.id }`. Clear activeCard.

ConfirmModal integration — show when `confirmPending` is set. Per-transition copy from UI-SPEC:

| From -> To                       | Title                    | Message                                                        | Confirm Label         |
| -------------------------------- | ------------------------ | -------------------------------------------------------------- | --------------------- |
| RC_PENDENTE -> RC_APROVADA       | "Aprovar requisicao?"    | "A requisicao {number} sera aprovada e seguira para cotacao."  | "Aprovar"             |
| RC_APROVADA -> EM_COTACAO        | "Iniciar cotacao?"       | "Uma solicitacao de cotacao sera criada a partir de {number}." | "Iniciar cotacao"     |
| EM_COTACAO -> OC_EMITIDA         | "Emitir pedido?"         | "A cotacao aprovada se tornara um pedido de compra formal."    | "Emitir pedido"       |
| OC_EMITIDA -> AGUARDANDO_ENTREGA | "Confirmar envio?"       | "O pedido {number} sera marcado como em transito."             | "Confirmar"           |
| AGUARDANDO_ENTREGA -> RECEBIDO   | "Registrar recebimento?" | "Voce sera redirecionado para confirmar os itens recebidos."   | "Ir para recebimento" |

On confirm: call `onCardMove(card.id, from, to)` + clear confirmPending.
On cancel: clear confirmPending (card returns to original column since no optimistic update yet).

Use ConfirmModal with `variant="warning"`.

DragOverlay: Render a `<KanbanCard card={activeCard} isOverlay />` clone.

Layout: horizontal flex with `overflow-x: auto`, 16px gap between columns.
