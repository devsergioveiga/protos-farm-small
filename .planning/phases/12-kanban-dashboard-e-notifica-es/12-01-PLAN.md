2. **Create `purchasing-kanban.service.ts`:**

Export function `getKanbanBoard(ctx: RlsContext, filters: KanbanFilters): Promise<KanbanBoard>`.

Implementation:

- Use `withRlsContext` from `../../database/rls`.
- Execute 5 parallel queries via `Promise.all`:
  - **getPurchaseRequestCards**: Split into RC_PENDENTE (status IN `RASCUNHO`, `PENDENTE`, `DEVOLVIDA`) and RC_APROVADA (status = `APROVADA` AND no related Quotation with `NOT EXISTS` subquery). Map to KanbanCard with entityType `'RC'`, sequentialNumber from `sequentialNumber`, urgency from `urgency`, requesterName from `createdBy -> User.name`, totalValue from sum of items, daysInStage = `Math.floor((Date.now() - updatedAt.getTime()) / 86400000)`, isOverdue = check if SLA exceeded (use slaDeadline field if exists, otherwise daysInStage > 7).
  - **getQuotationCards**: EM_COTACAO — Quotation where status IN (`AGUARDANDO_PROPOSTA`, `EM_ANALISE`). entityType `'SC'`, sequentialNumber from quotation, requesterName from RC -> createdBy -> User.name, totalValue from quotation estimatedTotal or sum of items.
  - **getPurchaseOrderCards**: Split into OC_EMITIDA (status IN `EMITIDA`, `CONFIRMADA`) and AGUARDANDO_ENTREGA (status = `EM_TRANSITO`). entityType `'OC'`, sequentialNumber from PO, requesterName from PO creator, isOverdue if expectedDeliveryDate < now.
  - **getGoodsReceiptCards**: RECEBIDO — GoodsReceipt where status = `CONFIRMADO` and related payable is not PAID. entityType `'GR'`.
  - **getPaidPayableCards**: PAGO — Payable where status = `PAID` AND paidAt >= 30 days ago, linked to a PurchaseOrder. entityType `'PAYABLE'`.

- Apply filters: if `filters.farmId`, add `farmId` WHERE clause to all queries. If `filters.urgency`, filter RC queries by urgency. If `filters.search`, filter by sequentialNumber LIKE.

- Build 7 columns from results: `[ { id: 'RC_PENDENTE', label: 'RC Pendente', count: rcPendingCards.length, cards: rcPendingCards }, ... ]`

3. **Create `purchasing-kanban.routes.ts`:**

```typescript
import { Router } from 'express';
import { authenticate } from '../../shared/auth/authenticate';
import { requirePermission } from '../../shared/auth/requirePermission';
import { getKanbanBoard } from './purchasing-kanban.service';

export const purchasingKanbanRouter = Router();

// GET /api/org/:orgId/purchasing/kanban
purchasingKanbanRouter.get(
  '/org/:orgId/purchasing/kanban',
  authenticate,
  requirePermission('purchases:read'),
  async (req, res, next) => {
    try {
      const ctx = { organizationId: req.params.orgId };
      const filters = {
        farmId: req.query.farmId as string | undefined,
        urgency: req.query.urgency as string | undefined,
        search: req.query.search as string | undefined,
      };
      const board = await getKanbanBoard(ctx, filters);
      res.json(board);
    } catch (err) {
      next(err);
    }
  },
);
```
