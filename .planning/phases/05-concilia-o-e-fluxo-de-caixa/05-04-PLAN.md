1. Create `useReconciliation.ts` hook file with:
   - `useImportHistory(bankAccountId?)` — fetches GET `/api/org/reconciliation/imports` with optional bankAccountId filter. Returns `{ imports, loading, error, refetch }`. Follow usePayables pattern for pagination.
   - `useImportLines(importId)` — fetches GET `/api/org/reconciliation/imports/${importId}/lines`. Returns `{ lines, loading, error, refetch }`. Lines are StatementLineWithMatches[].
   - `useReconciliationActions()` — returns action functions:
     - `uploadPreview(file: File): Promise<ImportPreviewResponse>` — POST to `/api/org/reconciliation/preview` with FormData (key 'file'). Use direct fetch() with FormData (not JSON api service — same pattern as CnabRetornoModal).
     - `confirmImport(file: File, bankAccountId: string, selectedLineIndices?: number[], columnMapping?): Promise<ImportResult>` — POST to `/api/org/reconciliation/imports` with FormData + JSON body fields.
     - `acceptMatch(importId, lineId, reconciliationId): Promise<void>` — POST to `/api/org/reconciliation/imports/${importId}/lines/${lineId}/confirm` with body `{reconciliationId}`.
     - `rejectMatch(importId, lineId, reconciliationId): Promise<void>` — POST to `/api/org/reconciliation/imports/${importId}/lines/${lineId}/reject`.
     - `manualLink(importId, lineId, links): Promise<void>` — POST to `/api/org/reconciliation/imports/${importId}/lines/${lineId}/link`.
     - `ignoreLine(importId, lineId): Promise<void>` — POST to `/api/org/reconciliation/imports/${importId}/lines/${lineId}/ignore`.
     - `searchCandidates(importId, search, bankAccountId): Promise<MatchCandidate[]>` — GET `/api/org/reconciliation/imports/${importId}/search?search=${search}&bankAccountId=${bankAccountId}`.
   - All fetches use auth token from existing api pattern.
