   - `parseOfxDate(dateStr: string): Date` ã parse `YYYYMMDDHHMMSS[offset:timezone]` format, strip timezone brackets if present
   - Limit: throw ReconciliationError if transactions.length > 5000

2. Create `csv-parser.ts`:
   - `detectCsvColumns(content: string): CsvDetectedColumns` ã read first 11 lines (1 header + 10 preview). Auto-detect separator (`;` or `,` or `\t`). Suggest column mapping by header name matching (data/date -> date, valor/amount/value -> amount, descricao/description/memo -> description, tipo/type -> type). Return headers array, suggestedMapping, and previewRows.
   - `parseCsv(content: string, mapping: CsvColumnMapping): ParsedStatementLine[]` ã parse all rows using confirmed mapping. Handle comma decimal separator: replace `,` with `.` after removing thousands dots. Detect CREDIT/DEBIT by sign (negative = DEBIT, positive = CREDIT). Limit: throw if > 5000 lines.
