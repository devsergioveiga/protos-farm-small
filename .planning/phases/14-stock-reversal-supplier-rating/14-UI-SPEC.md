## Component Inventory

### 1. Rating Badge (inline, QuotationModal)

**Scope:** QuotationModal supplier list items only — not SuppliersPage, not PurchaseOrderModal, not GoodsReceiptModal.

**Variants:**

| Condition                  | Badge Text          | Background            | Text Color            | Icon                                 |
| -------------------------- | ------------------- | --------------------- | --------------------- | ------------------------------------ |
| `rating < 2`               | "Avaliacao critica" | `--color-error-100`   | `--color-error-500`   | `AlertCircle` size=12, aria-hidden   |
| `rating >= 2 and < 3`      | "Avaliacao baixa"   | `--color-warning-100` | `--color-warning-500` | `AlertTriangle` size=12, aria-hidden |
| `rating >= 3` or no rating | no badge rendered   | —                     | —                     | —                                    |

**Badge specs:**

- Display: inline-flex, align-items center, gap 4px
- Padding: 2px 8px (vertical 2px, horizontal 8px)
- Border-radius: `--radius-full` (9999px) — pill shape
- Font: 14px Source Sans 3 weight 600
- Positioned: inline after supplier name span, before existing star display
- Tooltip on hover/focus: "Nota media: X.X (N avaliacoes)" — plain `title` attribute is acceptable for low-risk tooltip (no interactivity required)

**Supplier without any rating:**

- No badge, no stars (current behavior preserved — no change)
- Do not show "Sem avaliacao" text

**Accessibility:** Badge must not be the only indicator — color + icon + text (all three present).

Source: `14-CONTEXT.md` badge decision, `CLAUDE.md` color-not-alone rule, `QuotationModal.tsx` lines 290-297 (existing star slot)

### 2. Performance Tab (SupplierDetail / expanded supplier view)

**Where:** New "Performance" tab added to the supplier detail section in SuppliersPage. The tab panel sits alongside existing supplier info. Follow the existing Tabs pattern (tabs component with header/panel structure).

**Tab label:** "Performance"

**Tab panel layout (top to bottom):**

#### 2a. Period Filter Bar

- Single row, aligned right
- Four preset buttons: "Ultimo mes" | "Trimestre" | "Ano" | "Todos"
- Button style: ghost/outline, 14px Source Sans 3 semibold
- Active state: `--color-primary-600` border (2px solid), text `--color-primary-600`, background `--color-primary-50`
- Inactive state: `--color-neutral-200` border, text `--color-neutral-600`, background `--color-neutral-100`
- Touch target: minimum 48px height
- Spacing between buttons: 8px gap

#### 2b. Rating Trend Chart (LineChart)

- Title: "Evolucao da Nota" — 20px DM Sans bold, margin-bottom 16px
- Recharts `LineChart` inside `ResponsiveContainer width="100%" height={240}`
- XAxis: `dataKey="date"` formatted as "DD/MM/YY" — font Source Sans 3 12px, color `--color-neutral-500`
- YAxis: domain [1, 5], ticks [1, 2, 3, 4, 5] — font JetBrains Mono 12px, color `--color-neutral-500`
- CartesianGrid: `strokeDasharray="3 3"` color `--color-neutral-200`
- Line: `stroke="#2E7D32"` strokeWidth=2, dot r=4, activeDot r=6
- Tooltip: shows "Nota: X.X" for hovered point — no custom renderer needed
- Wrapper: `<figure>` with `<figcaption className="sr-only">Evolucao da nota do fornecedor ao longo do tempo</figcaption>`

#### 2c. Criteria Breakdown (horizontal bars)

- Title: "Nota por Criterio" — 20px DM Sans bold, margin-top 32px, margin-bottom 16px
- Four rows, one per criterion: "Prazo de Entrega", "Qualidade do Produto", "Preco", "Atendimento"
- Each row layout:
  - Label: 16px Source Sans 3 regular, min-width 160px, color `--color-neutral-700`
  - Bar: flex-grow 1, height 12px, background `--color-neutral-200`, border-radius `--radius-full`
  - Fill: width = (value/5 \* 100)%, background `--color-primary-500` (`#388E3C`), border-radius `--radius-full`
  - Value: 14px JetBrains Mono regular, min-width 40px, text-align right, color `--color-neutral-700`
- Row gap: 12px
- Accessible: each bar is `role="meter" aria-valuenow={value} aria-valuemin={1} aria-valuemax={5} aria-label="{criterion}: {value} de 5"`

#### 2d. Empty State (no ratings in selected period)

- Icon: `Star` size=48, color `--color-neutral-300`, aria-hidden
- Heading: "Nenhuma avaliacao neste periodo" — 20px DM Sans bold, color `--color-neutral-700`
- Body: "Registre avaliacoes apos cada entrega para acompanhar a performance deste fornecedor." — 16px Source Sans 3 regular, color `--color-neutral-500`, max-width 360px
- CTA button: "Avaliar fornecedor" — primary button style (`--color-primary-600` bg, white text)
- Layout: flex column, align-items center, gap 16px, padding-top 48px

#### 2e. Loading State (while fetching performance data)

- Show skeleton rows: three horizontal bar skeletons at 80%, 60%, 90% width
- Skeleton: `--color-neutral-200` background, animated opacity pulse (0.4→0.7, 1.5s infinite)
- No spinner, no full-page loading state

Source: `14-CONTEXT.md` performance report decisions, `MonthlyEvolutionChart.tsx` established Recharts pattern, `CLAUDE.md` skeleton/empty state rules

## Interaction States

### Rating Badge in QuotationModal

| State                  | Behavior                                                       |
| ---------------------- | -------------------------------------------------------------- |
| Hover badge            | `title` tooltip appears with "Nota media: X.X (N avaliacoes)"  |
| Focus badge (keyboard) | `title` tooltip + focus outline 2px `--color-primary-500`      |
| No rating              | Badge not rendered — no change to row layout                   |
| Selection              | Badge does not block checkbox interaction — informational only |

### Performance Tab Period Presets

| State               | Behavior                                                                |
| ------------------- | ----------------------------------------------------------------------- |
| Default on tab open | "Todos" selected                                                        |
| Click preset        | Instant filter update, new data fetch, loading skeleton during fetch    |
| Active button       | 2px solid `--color-primary-600` border, `--color-primary-50` background |
| Loading             | Skeleton bars replace chart content, preset buttons remain interactive  |

### Performance Tab Empty / Error

| State             | Behavior                                                       |
| ----------------- | -------------------------------------------------------------- |
| No data in period | Full empty state panel with CTA                                |
| CTA click         | Opens existing `SupplierRatingModal`                           |
| Fetch error       | Error message string (see copywriting), no retry button needed |
