    1. Create `apps/backend/src/modules/asset-acquisitions/asset-acquisitions.service.ts`:

       **Function: createAcquisitionAndPayable(ctx: RlsContext, input: CreateAssetAcquisitionInput)**
       - Validate: if `acquisitionValue > 0`, require `supplierId` (throw AssetAcquisitionError 400 "Fornecedor obrigatorio para aquisicao com valor")
       - Validate: if `paymentType === 'AVISTA'`, require `dueDate` (throw 400 "Data de vencimento obrigatoria para pagamento a vista")
       - Validate: if `paymentType === 'FINANCIADO'`, require `installmentCount >= 2` and `firstDueDate` (throw 400)
       - Resolve supplier name: `const supplier = await tx.supplier.findUnique({ where: { id: input.supplierId } })`
       - Use `prisma.$transaction(async (tx) => { ... })`:
         a. Create Asset via `tx.asset.create({ data: { ...assetFields, organizationId: ctx.organizationId } })`
            - Extract asset fields from input (name, type, classification, farmId, acquisitionValue, acquisitionDate, invoiceNumber, supplierId, costCenterId, costCenterMode, manufacturer, model, year, serialNumber, plate, etc.)
         b. If `acquisitionValue > 0` and `dueDate` or `firstDueDate`:
            - Determine installment params: AVISTA = 1 installment at dueDate; FINANCIADO = installmentCount installments at firstDueDate
            - Call `generateInstallments(Money(acquisitionValue), installmentCount, firstDueDate)` from `@protos-farm/shared`
            - Create Payable via `tx.payable.create`:
              - `organizationId: ctx.organizationId`
              - `farmId: input.farmId`
              - `supplierName: supplier.name`
              - `category: 'ASSET_ACQUISITION'`
              - `description: 'Aquisicao ${asset.assetTag} — ${asset.name}'`
              - `totalAmount: Money(acquisitionValue).toDecimal()`
              - `dueDate: installments[0].dueDate`
              - `documentNumber: input.invoiceNumber ?? null`
              - `installmentCount`
              - `originType: 'ASSET_ACQUISITION'`
              - `originId: asset.id`
            - Create PayableInstallments via `tx.payableInstallment.createMany`
            - If costCenterId: create PayableCostCenterItem via `tx.payableCostCenterItem.create` with `allocMode: 'PERCENTAGE'`, `percentage: 100`
         c. Return `{ asset: { id, assetTag, name }, payableId, installmentCount }`
       - CRITICAL: Do NOT call `payables.service.createPayable()` — it uses `withRlsContext` which creates a nested transaction and will deadlock. Use `tx.payable.create` directly, same pattern as goods-receipts.service.ts.

       **Function: parseNfeUpload(buffer: Buffer)**
       - Convert buffer to UTF-8 string
       - Call `parseNfeXml(xmlString)` from nfe-parser.ts
       - Validate: if no items found, throw AssetAcquisitionError 400 "Nenhum item encontrado no XML"
       - Return NfeParsedData

       **Function: createFromNfe(ctx: RlsContext, nfeParsed: NfeParsedData, input: CreateFromNfeInput)**
       - Call `calculateRateio(nfeParsed.items, freight, insurance, otherCosts)` to get per-item acquisitionValues
       - Use `prisma.$transaction(async (tx) => { ... })`:
         a. For each item assignment in input.items:
            - If `existingAssetId`: update existing asset with acquisitionValue
            - Else: create new Asset with assetName, assetType, classification, farmId, acquisitionValue from rateio
         b. Create single Payable for total NF amount (same pattern as createAcquisitionAndPayable)
         c. Create installments based on paymentType
         d. Create PayableCostCenterItem if costCenterId provided
       - Return NfeAcquisitionOutput
