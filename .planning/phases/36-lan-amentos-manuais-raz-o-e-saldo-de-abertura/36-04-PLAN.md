    **Hook** (`hooks/useJournalEntries.ts`):

    Use the existing `api` service for HTTP calls. Provide:
    - `useJournalEntries(orgId, filters?)` — GET `/api/org/${orgId}/journal-entries` with query params; returns `{ entries, loading, error, refetch }`
    - `useJournalEntry(orgId, entryId)` — GET single entry with lines
    - `createDraft(input)` — POST to create draft
    - `postEntry(entryId)` — POST `/:id/post`
    - `reverseEntry(entryId, reason)` — POST `/:id/reverse`
    - `deleteDraft(entryId)` — DELETE `/:id`
    - `saveTemplate(input)` — POST `/templates`
    - `listTemplates()` — GET `/templates`
    - `deleteTemplate(templateId)` — DELETE `/templates/:id`

    **Hook** (`hooks/useOpeningBalance.ts`):

    - `useOpeningBalancePreview(orgId, fiscalYearId)` — GET `/api/org/${orgId}/opening-balance/preview/${fiscalYearId}`; returns `{ lines, loading, error }`
    - `postOpeningBalance(input)` — POST `/api/org/${orgId}/opening-balance`
