**Create `payroll-engine.service.ts`:**

```typescript
import Decimal from 'decimal.js';
import { Parser } from 'expr-eval';
```

Set `Decimal.set({ rounding: Decimal.ROUND_HALF_UP });` at module level.

**`calculateINSS(grossSalary: Decimal, brackets: INSSBracket[], ceiling: Decimal): INSSResult`**
- Cap effectiveBase at min(grossSalary, ceiling)
- Loop through brackets in order
- For each bracket, calculate bracket width = upTo ? (upTo - from + 0.01) : remaining
- applicable = min(remaining, bracketWidth)
- contribution += applicable * rate
- remaining -= applicable
- Break when remaining <= 0
- Return contribution rounded to 2 decimal places, effectiveRate = contribution / effectiveBase

**`calculateIRRF(input: IRRFInput): IRRFResult`**
- Compute taxableBase = grossSalary - inssContribution - (dependents * dependentDeduction) - alimony
- If taxableBase <= 0, return all zeros
- Step 1: Find correct bracket (where upTo is null or taxableBase <= upTo), calculate grossTax = taxableBase * rate - deduction
- Step 2: If taxableBase <= exemptionLimit (5000), return finalTax = 0
- If taxableBase <= redutorUpperLimit (7350), compute redutor = redutorA - (redutorB * taxableBase), finalTax = max(0, grossTax - redutor)
- Else finalTax = grossTax
- Round all results to 2 decimal places

**`calculateFGTS(grossSalary: Decimal): FGTSResult`**
- contribution = grossSalary * 0.08, rounded to 2 places
- No ceiling

**`calculateSalaryFamily(input: SalaryFamilyInput): SalaryFamilyResult`**
- If grossSalary > incomeLimit, return {benefit: 0, eligible: false}
- benefit = eligibleDependents * valuePerChild, rounded to 2 places

**`calculateRuralNightPremium(input: RuralNightInput): RuralNightResult`**
- premium = nightHours * hourlyRate * premiumRate, rounded to 2 places
- Note: rural hours are 60 minutes (caller passes actual hours, not converted)

**`calculateRuralUtilityDeductions(input: RuralUtilityInput): RuralUtilityResult`**
- housingCap = regionalMinWage * 0.20
- foodCap = regionalMinWage * 0.25
- housing = min(requestedHousing, housingCap)
- food = min(requestedFood, foodCap)
- housingCapped = requestedHousing > housingCap
- foodCapped = requestedFood > foodCap

**`evaluateFormula(formula: string, context: RubricaContext): Decimal`**
- Use `new Parser()` from expr-eval
- `parser.evaluate(formula, context)`
- Return result as Decimal, rounded to 2 places
- If formula is empty or null, return Decimal(0)
- If evaluation throws, throw a descriptive error (FormulaEvaluationError)

Export all functions as named exports.
