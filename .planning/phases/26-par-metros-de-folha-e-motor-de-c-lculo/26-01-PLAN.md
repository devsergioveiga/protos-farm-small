- `isActive Boolean @default(true)`
- `eSocialCode String?`
- `createdBy String`
- `createdAt DateTime @default(now())`

- `id String @id @default(uuid())`

- `notes String?`
- `createdBy String`

- Map: `@@map("payroll_legal_tables")`

- `id String @id @default(uuid())`
- `tableId String`
- `fromValue Decimal @db.Decimal(12, 2)`

- `order Int`
- Relation: `table PayrollLegalTable @relation(fields: [tableId], references: [id], onDelete: Cascade)`

- Map: `@@map("payroll_table_brackets")`

- `id String @id @default(uuid())`
