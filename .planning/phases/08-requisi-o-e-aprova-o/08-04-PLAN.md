**3. Create `apps/frontend/src/hooks/usePurchaseRequestForm.ts`:**

Following useSupplierForm.ts pattern:

- `create(input)` — POST `/api/org/purchase-requests`, returns created RC
- `update(id, input)` — PUT `/api/org/purchase-requests/${id}`, returns updated RC
- `remove(id)` — DELETE `/api/org/purchase-requests/${id}`
- `submit(id)` — POST `/api/org/purchase-requests/${id}/transition` with body `{ action: 'SUBMIT' }`
- `transition(id, input)` — POST `/api/org/purchase-requests/${id}/transition` with body input
- `uploadAttachment(id, file)` — POST `/api/org/purchase-requests/${id}/attachments` with FormData
- Call `onSuccess()` callback after each mutation
- Return: `{ create, update, remove, submit, transition, uploadAttachment, isLoading, error }`
