**`createPurchaseRequest(ctx, input)`**:

- Validate: `input.requestType` must be in `RC_TYPES`, `input.urgency` must be in `RC_URGENCY_LEVELS`
- Validate: `input.items.length >= 1` — throw PurchaseRequestError('Adicione pelo menos um item a requisicao.', 400)

- Inside `withRlsContext(ctx, async (tx) => { ... })`:
  - Call `getNextSequentialNumber(tx, ctx.organizationId)`
  - `tx.purchaseRequest.create({ data: { organizationId: ctx.organizationId, farmId: input.farmId, sequentialNumber, requestType: input.requestType, urgency: input.urgency, status: 'RASCUNHO', justification: input.justification, costCenterId: input.costCenterId, neededBy: input.neededBy ? new Date(input.neededBy) : null, geolat: input.geolat, geolon: input.geolon, photoUrl: input.photoUrl, createdBy: ctx.userId, items: { create: input.items.map(i => ({ productId: i.productId, productName: i.productName, quantity: i.quantity, unitId: i.unitId, unitName: i.unitName, estimatedUnitPrice: i.estimatedUnitPrice, notes: i.notes })) } }, include: { items: true, farm: { select: { id: true, name: true } } } })`

**`getPurchaseRequestById(ctx, id)`**:

- `withRlsContext` query with `include: { items: true, attachments: true, approvalActions: { include: { assignee: { select: { id: true, name: true } } }, orderBy: { step: 'asc' } }, farm: { select: { id: true, name: true } }, creator: { select: { id: true, name: true } }, costCenter: { select: { id: true, name: true } } }`

**`listPurchaseRequests(ctx, query)`**:

- Default page=1, limit=20, max 100
- Build `where` clause: `organizationId: ctx.organizationId`, `deletedAt: null`

**1. Create `apps/backend/src/modules/purchase-requests/purchase-requests.routes.ts`:**

Express Router with 6 endpoints, following suppliers.routes.ts pattern:

- `POST /` — `checkPermission('purchases:manage')`, validates body, calls `createPurchaseRequest(ctx, body)`, returns 201
- `GET /` — `checkPermission('purchases:read')`, parses query params (page, limit, status, search, farmId, urgency, createdBy), calls `listPurchaseRequests(ctx, query)`, returns 200
- `GET /:id` — `checkPermission('purchases:read')`, calls `getPurchaseRequestById(ctx, req.params.id)`, returns 200
- `PUT /:id` — `checkPermission('purchases:manage')`, validates body, calls `updatePurchaseRequest(ctx, req.params.id, body)`, returns 200
- `DELETE /:id` — `checkPermission('purchases:manage')`, calls `deletePurchaseRequest(ctx, req.params.id)`, returns 204
- `POST /:id/attachments` — `checkPermission('purchases:manage')`, uses multer single file upload (`upload.single('file')`), saves to `uploads/purchase-requests/{orgId}/{rcId}/`, creates PurchaseRequestAttachment row, returns 201

Error handling: catch PurchaseRequestError and return `{ error: err.message }` with `err.statusCode`. Unknown errors: log and return 500 `{ error: 'Erro interno do servidor' }`.

Configure multer disk storage:

```typescript
const storage = multer.diskStorage({
  destination: (req, file, cb) => {
    const dir = path.join('uploads', 'purchase-requests', req.params.orgId || 'unknown');
    fs.mkdirSync(dir, { recursive: true });
    cb(null, dir);
  },
  filename: (req, file, cb) => cb(null, `${Date.now()}-${file.originalname}`),
});
const upload = multer({ storage, limits: { fileSize: 10 * 1024 * 1024 } }); // 10MB
```
