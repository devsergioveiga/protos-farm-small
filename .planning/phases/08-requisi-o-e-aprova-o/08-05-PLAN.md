**1. Create `useApprovalRules.ts`:**

Hook for approval rules CRUD + delegations:

- `listRules()` — GET `/api/org/approval-rules`, returns rules ordered by priority
- `createRule(input)` — POST `/api/org/approval-rules`
- `updateRule(id, input)` — PUT `/api/org/approval-rules/${id}`
- `deleteRule(id)` — DELETE `/api/org/approval-rules/${id}`
- `reorderRules(orderedIds)` — POST `/api/org/approval-rules/reorder` with body `{ orderedIds }`
- `listDelegations()` — GET `/api/org/approval-rules/delegations`
- `createDelegation(input)` — POST `/api/org/approval-rules/delegations`
- `deactivateDelegation(id)` — PATCH `/api/org/approval-rules/delegations/${id}/deactivate`
- Return: `{ rules, delegations, isLoading, createRule, updateRule, deleteRule, reorderRules, createDelegation, deactivateDelegation, refresh }`

**1. Create `useNotifications.ts`:**

Hook for notification bell:

- State: `notifications` array, `unreadCount` number, `isOpen` boolean
- `fetchNotifications()` — GET `/api/org/notifications?unread=true&limit=20`
- `fetchUnreadCount()` — GET `/api/org/notifications/unread-count`
- `markAsRead(id)` — PATCH `/api/org/notifications/${id}/read`
- `markAllAsRead()` — PATCH `/api/org/notifications/read-all`
- Polling: `useEffect` with `setInterval(fetchUnreadCount, 30000)` — 30 second interval. No loading indicator during background poll.
- On mount: fetch both notifications and unread count
- Return: `{ notifications, unreadCount, isOpen, setIsOpen, markAsRead, markAllAsRead, refresh }`
