2. **Create transfers.service.ts** with functions:
- `createTransfer(ctx, input)`: Inside withRlsContext transaction: validate both accounts exist and are active, validate fromAccountId !== toAccountId, create AccountTransfer record, create 2 FinancialTransactions (DEBIT on from, CREDIT on to) with referenceType 'TRANSFER' and referenceId = transfer.id, decrement from BankAccountBalance, increment to BankAccountBalance. If feeAmount provided and > 0: create 3rd FinancialTransaction with referenceType 'TRANSFER_FEE', decrement from BankAccountBalance by fee. Use Money() for all amounts.
- `listTransfers(ctx, query)`: Query with optional filters (date range via transferDate, type, accountId filtering on fromAccountId OR toAccountId). Include fromAccount.name and toAccount.name. Return array of TransferOutput.
- `getTransfer(ctx, transferId)`: Find by id + organizationId, include fromAccount + toAccount. Return TransferOutput or throw 404.
- `deleteTransfer(ctx, transferId)`: Find transfer, reverse the balance changes (increment from, decrement to, add fee back if exists), delete FinancialTransactions with referenceId, delete transfer. All in withRlsContext.
