**Create `apps/frontend/src/hooks/useGoodsReceipts.ts`:**

Two hooks following the usePurchaseOrders pattern:

```typescript
// useGoodsReceipts(filters) — paginated list with refetch
// usePendingDeliveries() — single-fetch list of POs awaiting delivery
```

`useGoodsReceipts` hook:

- Params: { page?: number; limit?: number; status?: string; search?: string; receivingType?: string }
- State: goodsReceipts (GoodsReceiptListItem[]), total, totalPages, isLoading, error, refetchCounter
- useEffect: build URLSearchParams, call api.get<{ data, total, page, totalPages }>('/org/goods-receipts?...')
- refetch: () => setRefetchCounter(c => c + 1)
- Return { goodsReceipts, total, totalPages, isLoading, error, refetch }

`usePendingDeliveries` hook:

- No params
- State: deliveries (PendingDelivery[]), isLoading, error
- useEffect: call api.get<PendingDelivery[]>('/org/goods-receipts/pending')
- Return { deliveries, isLoading, error, refetch }

Additional API functions (exported, not hooks — for modal use in Plan 05):

```typescript
export async function createGoodsReceiptApi(input: CreateGoodsReceiptInput): Promise<GoodsReceipt> {
  return api.post('/org/goods-receipts', input);
}
export async function getGoodsReceiptApi(id: string): Promise<GoodsReceipt> {
  return api.get(`/org/goods-receipts/${id}`);
}
export async function transitionGoodsReceiptApi(
  id: string,
  status: string,
  rejectionReason?: string,
): Promise<GoodsReceipt> {
  return api.put(`/org/goods-receipts/${id}/transition`, { status, rejectionReason });
}
export async function confirmGoodsReceiptApi(id: string): Promise<GoodsReceipt> {
  return api.put(`/org/goods-receipts/${id}/confirm`);
}
```
