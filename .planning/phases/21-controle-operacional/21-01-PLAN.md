       - Import `Decimal` from `decimal.js` and prisma from `../../database/prisma`

       - Export `interface OperationalCostResult` with fields: `acquisitionValue: number`, `accumulatedDepreciation: number`, `netBookValue: number`, `maintenanceCost: number`, `fuelCost: number`, `insuranceCost: null`, `totalOperationalCost: number`, `totalLifetimeCost: number`, `costPerHour: number | null`, `currentHourmeter: number | null`, `fuelRecordCount: number`, `notes: string[]`
       - Export `async function getOperationalCost(ctx: { organizationId: string }, assetId: string, periodStart?: string, periodEnd?: string): Promise<OperationalCostResult>`
       - Inside: find asset with `prisma.asset.findFirst({ where: { id: assetId, organizationId: ctx.organizationId, deletedAt: null }, select: { acquisitionValue: true, currentHourmeter: true, currentOdometer: true, assetType: true } })`. Throw OperationalCostError('Ativo não encontrado', 404) if null.
       - Build `dateFilter` object: if periodStart and periodEnd provided, use `{ gte: new Date(periodStart), lte: new Date(periodEnd) }`; if only periodStart, `{ gte: new Date(periodStart) }`; if only periodEnd, `{ lte: new Date(periodEnd) }`; otherwise undefined.
       - Aggregate accumulated depreciation: `prisma.depreciationEntry.aggregate({ where: { assetId, organizationId: ctx.organizationId, reversedAt: null }, _sum: { depreciationAmount: true } })` — depreciation is cumulative, NOT filtered by period.
       - Aggregate maintenance cost: `prisma.workOrder.aggregate({ where: { assetId, organizationId: ctx.organizationId, status: 'CONCLUIDA', ...(dateFilter ? { completedAt: dateFilter } : {}) }, _sum: { totalCost: true } })` — nullable field, guard with `?? 0`.
       - Aggregate fuel cost: `prisma.fuelRecord.aggregate({ where: { assetId, organizationId: ctx.organizationId, ...(dateFilter ? { fuelDate: dateFilter } : {}) }, _sum: { totalCost: true }, _count: { id: true } })`.
       - Use `new Decimal(String(value ?? 0))` for ALL numeric conversions — NEVER native JS arithmetic.
       - Compute: `acquisitionValue = new Decimal(String(asset.acquisitionValue ?? 0))`, `depreciationTotal`, `maintenanceTotal`, `fuelTotal`.
       - `netBookValue = acquisitionValue.minus(depreciationTotal)` (floor at 0 with `.max(new Decimal(0))`).
       - `totalOperationalCost = maintenanceTotal.plus(fuelTotal)`.
       - `totalLifetimeCost = acquisitionValue.plus(maintenanceTotal).plus(fuelTotal)`.
       - `costPerHour`: if `asset.currentHourmeter` is not null and > 0, `totalLifetimeCost.div(new Decimal(String(asset.currentHourmeter))).toDecimalPlaces(2).toNumber()`, else null.
       - `notes` array: always push `"Custo de seguro não disponível — campo não modelado"` (insurance is null per research decision).
       - Return object with all `.toNumber()` conversions.

       - Call `getOperationalCost(ctx, req.params.assetId, periodStart, periodEnd)`.
