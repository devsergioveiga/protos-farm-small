1. Create `apps/backend/src/modules/assets/assets.service.ts`:

**getNextAssetTag(tx, organizationId):**
- Query last asset tag: `tx.asset.findFirst({ where: { organizationId, assetTag: { startsWith: 'PAT-' } }, orderBy: { assetTag: 'desc' }, select: { assetTag: true } })`
- Parse number from `PAT-NNNNN` format, increment, pad to 5 digits
- Return `PAT-${String(lastNum + 1).padStart(5, '0')}`
- First tag: `PAT-00001`

**createAsset(ctx, input):**
- Wrap in `prisma.$transaction`
- Generate assetTag via getNextAssetTag
- Validate: if assetType is IMPLEMENTO and parentAssetId is provided, verify parent exists and is type MAQUINA
- Validate: if assetType is TERRA, force classification to NON_DEPRECIABLE_CPC27
- Validate: if assetType is BENFEITORIA and geoLat+geoLon provided, build raw SQL for PostGIS point insertion: `ST_SetSRID(ST_MakePoint(lon, lat), 4326)`
- Use `Decimal` from decimal.js for acquisitionValue
- Return created asset with include: { farm: { select: { name: true } }, supplier: { select: { name: true } }, costCenter: { select: { name: true } }, parentAsset: { select: { id: true, name: true, assetTag: true } } }

**listAssets(ctx, query):**
- Build where clause: organizationId, deletedAt: null, optional filters for farmId, assetType, status, search (OR on name/assetTag/serialNumber with mode insensitive), minValue/maxValue on acquisitionValue, acquisitionFrom/acquisitionTo on acquisitionDate
- Paginate with skip/take (default page=1, limit=20)
- Include farm.name, supplier.name, costCenter.name
- Return { data, total, page, limit, totalPages }

**getAsset(ctx, assetId):**
- Find by id + organizationId + deletedAt: null
- Include farm, supplier, costCenter, parentAsset, childAssets, _count for fuelRecords/meterReadings/documents
- Throw AssetError 404 if not found

**updateAsset(ctx, assetId, input):**
- Find existing, throw 404 if not found
- Apply same validations as create (parentAssetId check, TERRA classification)
- Handle geoPoint update via raw SQL if geoLat/geoLon changed
- Return updated asset

**deleteAsset(ctx, assetId):**
- Soft delete: set deletedAt = new Date()
- Return updated asset

**getAssetSummary(ctx):**
- Aggregate queries: count total, sum acquisitionValue, count where status=EM_MANUTENCAO, last 5 acquired (orderBy acquisitionDate desc)
- Return { totalAssets, totalValue, inMaintenance, recentAcquisitions[] }

**uploadAssetPhoto(ctx, assetId, filePaths):**
- Find asset, throw 404 if not found
- Append new file paths to existing photoUrls array
- Return updated photoUrls

**removeAssetPhoto(ctx, assetId, photoUrl):**
- Find asset, filter out photoUrl from array
- Delete file from disk if exists
- Return updated photoUrls
