# Protos Farm — Design System

## Filosofia de Design

**"Terra digital"** — Uma interface que traduz a solidez e organicidade do campo em experiência digital. Robusta como solo fértil, clara como céu aberto, funcional como ferramenta de trabalho.

### Princípios Fundamentais

| #   | Princípio                   | Descrição                                                                                                                            |
| --- | --------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| 1   | **Clareza Brutal**          | Informação sem ruído. Agricultores operam sob sol forte, com mãos sujas e pressa. Cada pixel precisa justificar sua existência.      |
| 2   | **Confiança Visual**        | Cores sólidas, tipografia firme, feedback imediato. O sistema deve transmitir que os dados estão seguros e as ações são previsíveis. |
| 3   | **Offline-First Mindset**   | Zona rural = conexão instável. UI deve comunicar estado de sincronização e funcionar com dados em cache.                             |
| 4   | **Toque Generoso**          | Áreas de toque mínimas de 48px. Botões grandes. Formulários espaçados. Operadores usam luvas, telas empoeiradas, dedos calejados.    |
| 5   | **Hierarquia Imediata**     | O dado mais importante aparece primeiro e maior. Sem scroll para achar o essencial. Dashboard = glance value.                        |
| 6   | **Cross-Platform Coerente** | Web e mobile compartilham tokens, lógica visual e componentes conceituais. Código difere, identidade não.                            |

## Tom & Personalidade

- **Profissional**, mas não corporativo
- **Orgânico**, mas não infantil
- **Direto**, sem jargão técnico desnecessário
- **Acolhedor**, especialmente em estados de erro
