2. **Update financial-dashboard.service.ts** — add computed fields AFTER totalBankBalance calculation:

```typescript
// ─── Accounting Balance (saldo contabil) ─────────────────────────
import { getAccountingBalanceData } from '../checks/checks.service';
// Inside the withRlsContext block, after totalBankBalance:

const { pendingEmitidos, pendingRecebidos } = await getAccountingBalanceData(ctx, farmId);
const accountingBalance = Money(totalBankBalance)
  .subtract(Money(pendingEmitidos))
  .add(Money(pendingRecebidos))
  .toNumber();

// ─── Open Bills Count ─────────────────────────────────────────────
const openBillsCount = await (tx as any).creditCardBill.count({
  where: {
    organizationId: ctx.organizationId,
    status: 'OPEN',
    expenses: { some: {} }, // only count bills that have expenses
    ...(farmId ? { creditCard: { farmId } } : {}),
  },
});

// ─── Checks Near Compensation (7 days) ────────────────────────────
const sevenDaysFromNow = new Date(todayUtc.getTime() + 7 * 86400000);
const checksNearCompensation = await (tx as any).check.count({
  where: {
    organizationId: ctx.organizationId,
    status: 'A_COMPENSAR',
    expectedCompensationDate: { gte: todayUtc, lte: sevenDaysFromNow },
    ...(farmId ? { bankAccount: { farms: { some: { farmId } } } } : {}),
  },
});
```

Add these 5 new fields to the return object: `accountingBalance, pendingEmitidos, pendingRecebidos, openBillsCount, checksNearCompensation`.
