2. **Create credit-cards.service.ts** with functions:

- **`getBillPeriod(closingDay: number, referenceDate: Date)`**: Helper. periodStart = closingDay of PREVIOUS month (UTC). periodEnd = closingDay - 1 of CURRENT month (23:59:59.999 UTC). If closingDay - 1 === 0, use last day of previous month. Returns `{ start: Date, end: Date, dueDate: Date }` where dueDate = dueDay of current month.

- **`getOrCreateBill(tx, organizationId, creditCardId, closingDay, dueDay, referenceDate)`**: Helper. Calculate period via getBillPeriod. Find existing OPEN bill for that period (match by creditCardId + periodStart). If not found, create new bill with status OPEN, totalAmount 0. Return bill.

- **`createCreditCard(ctx, input)`**: Validate closingDay 1-28, dueDay 1-28. Validate debitAccountId exists and isActive. Validate farmId exists. Create CreditCard record. Return CreditCardOutput.

- **`listCreditCards(ctx)`**: List active cards with debitAccount name, farm name, and current OPEN bill (include expenses). Return CreditCardOutput[].

- **`getCreditCard(ctx, cardId)`**: Get card with all bills (last 3) including expenses. Return with debitAccount and farm names.

- **`updateCreditCard(ctx, cardId, input)`**: Update card fields. Validate closingDay/dueDay if changed.

- **`deleteCreditCard(ctx, cardId)`**: Soft delete (set isActive=false). Only if no OPEN bills with expenses.

- **`addExpense(ctx, cardId, input)`**: Inside withRlsContext: validate card exists and isActive. Split amount: `installmentAmount = Money(input.amount).divide(input.totalInstallments)` with ROUND_DOWN on base and residual on first installment. For each installment i=0..totalInstallments-1: calculate referenceDate = expenseDate + i months (UTC month arithmetic), call getOrCreateBill for that period, create CreditCardExpense with billId, installmentNumber=i+1, amount=installmentAmount (or residual for i===0). Update each bill's totalAmount by incrementing.

- **`listBills(ctx, cardId, limit=3)`**: List bills for card ordered by periodStart DESC. Include expenses.

- **`closeBill(ctx, billId)`**: Inside withRlsContext: find bill with status OPEN, include creditCard + expenses. If not found throw 404. If expenses.length === 0 throw 422 "Adicione ao menos um gasto antes de fechar a fatura." Calculate total from expenses using Money. Import and call `createPayable(ctx, { farmId: bill.creditCard.farmId, supplierName: bill.creditCard.name, category: 'CARTAO_CREDITO', description: 'Fatura {cardName} — venc. {dueDate}', totalAmount: total.toNumber(), dueDate: bill.dueDate.toISOString(), bankAccountId: bill.creditCard.debitAccountId, installmentCount: 1, costCenterItems: [] })`. Update bill: status='CLOSED', payableId=payable.id, closedAt=new Date(). Return payable.

- **`getOpenBillsCount(ctx)`**: Count OPEN bills across all cards. Return `{ count: number }`.
